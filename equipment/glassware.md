# 酒杯完全指南 (Glassware)

> 酒杯不是玄学，是有科学依据的功能选择。
> 合适的杯具影响香气、口感、温度、甚至饮酒的心情。

---

## 一、基础原则

### 1.1 为什么杯具很重要
```
功能三重奏：
  1. 香气收集：杯口收窄 → 香气集中在杯口 → 闻香更清晰
  2. 温度控制：杯壁薄 → 冰镇更有效；杯壁厚 → 保温更好
  3. 视觉呈现：合适的酒配合适的杯 → 颜值即正义

杯具容量选择：
  - 酒液占杯 1/3-1/2 → 最优雅（留空间给香气）
  - 不要倒满，倒满 = 没留香气的空间
```

### 1.2 容量速查
| 杯型 | 容量 | 实际盛酒量 | 适用 |
|------|------|-----------|------|
| Coupe | 150-180ml | 90-120ml | 短饮（无冰/小批量）|
| Nick & Nora | 180-200ml | 120-150ml | 现代经典替代Coupe |
| 马天尼杯 | 180-240ml | 90-150ml | Martini、Cosmopolitan |
| 岩石杯 (Rocks) | 180-300ml | 90-180ml | 古典、加冰 |
| 高球杯 (Highball) | 250-350ml | 200-300ml | Highball类 |
| 柯林斯杯 (Collins) | 300-350ml | 240-300ml | Collins类、Mojito |
| 古典杯 | 180-300ml | 90-180ml | 加冰古典 |
| 白兰地杯 | 300-600ml | 30-90ml（少量）| 白兰地/干邑 |
| 葡萄酒杯 | 300-600ml | 120-180ml | 加强葡萄酒 |

## 二、经典杯型详解

### 2.1 马天尼杯 (Martini Glass)
```
特征：倒三角，浅宽杯身，长柄
容量：180-240ml
标准盛酒量：90-150ml（只倒至杯身 1/2）

适用酒款：
  - Dry Martini
  - Cosmopolitan
  - Manhattan（变体）
  - 花式短饮

为什么用它：
  - V 字形让香气集中在顶部
  - 浅杯身让视觉欣赏酒色
  - 长柄避免手温传导

注意事项：
  - 必须冰镇（杯壁薄，冷得快，热得也快）
  - 不要用它装太满（视觉+香气双重灾难）
```

### 2.2 Coupe 杯 (Coup Glass)
```
特征：浅弧形杯身，比马天尼更圆
容量：150-180ml
趋势：正在替代马天尼杯成为新经典

适用酒款：
  - Daiquiri
  - Sidecar
  - Aviation
  - 任何酸味短饮

优点：
  - 比马天尼杯更稳（不容易翻倒）
  - 更适合有泡沫的酒
  - 外观优雅

缺点：
  - 香气收集不如马天尼
  - 手温容易影响酒液
```

### 2.3 Nick & Nora 杯
```
特征：Coupe 和马天尼的结合体，椭圆口
容量：180-200ml
发明：1930年代，为解决马天尼杯易翻倒问题

适用：正在成为现代鸡尾酒的标准杯
  - 几乎所有酸味短饮
  - 替代 Coupe 成为新经典

优点：
  - 容量适中（比 Coupe 大）
  - 形状更有设计感
  - 适合拍照
```

### 2.4 岩石杯 (Rocks Glass)
```
特征：矮身、厚壁、宽口
容量：180-300ml
别名：古典杯、老式杯

适用酒款：
  - Old Fashioned（必须）
  - Negroni（可）
  - 威士忌纯饮
  - 任何加方冰的酒

杯壁厚度：
  - 薄壁：现代简洁
  - 厚壁：传统经典（保冷更好）

加分细节：杯底加厚（Heavy Bottom）更稳
```

### 2.5 高球杯 (Highball Glass)
```
特征：高而细长，直壁或微锥
容量：250-350ml
适用：大量液体+冰的鸡尾酒

适用酒款：
  - Gin & Tonic
  - Moscow Mule
  - Rum & Coke
  - Screwdriver
  - 长饮类

为什么用高球：
  - 容纳大量液体
  - 直壁让冰块容易堆积
  - 长形状让气泡保持更久
```

### 2.6 柯林斯杯 (Collins Glass)
```
特征：比高球更细长，容量更大
容量：300-350ml
适用：需要更多气泡和空间的酒

与高球杯的区别：
  - 柯林斯更细更高
  - 高球更宽更稳
  - 两者经常互换使用

适用酒款：
  - Tom Collins
  - Mojito（柯林斯或高球）
  - Singapore Sling（派尔森杯或柯林斯）
```

### 2.7 白兰地杯 / 干邑杯 (Snifter)
```
特征：球形杯身，收口，大肚窄口
容量：300-600ml
盛酒量：通常只倒 30-90ml（1/4 杯）

适用酒款：
  - 白兰地/干邑（纯饮）
  - 雅文邑
  - 白兰地古典（加方冰）

为什么这样设计：
  - 大肚让酒与空气接触（醒酒）
  - 窄口收集香气
  - 手握杯肚温暖酒液（适合不需冰的酒）
```

### 2.8 红酒杯（鸡尾酒应用）
```
适用场景：
  - 热饮酒（红酒杯保温）
  - 有些酒吧用它代替高球（更大气）
  - 加蛋清的酸味酒（容量够）

选杯建议：
  - 白葡萄酒杯（中型）：适合大多数酸味短饮
  - 红葡萄酒杯（大）：适合古典鸡尾酒
```

## 三、特殊杯型

### 3.1 铜马克杯 (Copper Mug)
```
用途：Moscow Mule（莫斯科骡子）
为什么用铜：
  - 铜是极好的导热体
  - 手握铜杯时，金属快速传热 → 手感凉爽
  - 视觉效果（经典搭配）

替代：不锈钢马克杯（次选）
```

### 3.2 Julep 杯 (Mint Julep Cup)
```
特征：银/不锈钢，柱形或杯形，厚壁
容量：200-300ml
用途：Mint Julep（必须）

为什么用银/不锈钢：
  - 极冷触感（适合 Julep 大量碎冰）
  - 传统（肯塔基赛马会标配）
  - 银的光泽增加奢华感

替代：玻璃 Julep 杯（便宜很多）
```

### 3.3 子弹杯 (Shot Glass)
```
容量：30-60ml
用途：
  - B-52（分层 Shot）
  - 短饮Shot（快速出品）
  - 测量工具（用量杯代替）

不建议：用来做"一口闷"的烈酒shot（对酒不尊重）
```

### 3.4 披萨杯 / 小杯 (Pony Glass)
```
容量：60-120ml
用途：
  - Pony（少量鸡尾酒）
  - 小份甜点酒
  - 替代子弹杯做精致Shot

不如：Nick & Nora（更优雅）
```

## 四、杯具购买建议

### 4.1 家庭入门套装
```
推荐套装（6-8只）：
  - 马天尼杯 × 2

  - 岩石杯 × 2
```

## 五、杯具保养

```
清洗：
  - 玻璃杯：手洗更佳（洗碗机可能造成磨损）
  - 薄壁杯：冷水+温和清洁剂，不要热水
  - 铜杯/银杯：专用清洁剂，避免氧化

储存：
  - 倒扣在杯架上
  - 或用干净的毛巾覆盖
  - 避免堆叠（可能磕碰）

使用前：
  - 放入冷冻室 15-30 分钟
  - 或用冰水冲洗
  - 确保无水渍无指纹

水渍处理：
  - 白醋+水（1:1）浸泡
  - 或专业玻璃清洁剂
```

```
酒款 → 正确杯具：

```

## 七、特殊材质

```
普通玻璃 vs 水晶玻璃：
  - 水晶（含铅）→ 更亮、更薄、声音清脆
  - 无铅水晶 → 替代选项（便宜）
  - 普通玻璃 → 经济实惠，耐摔

不锈钢杯：
  - 用于户外/泳池派对
  - 不怕摔，但影响香气感知

亚克力(Plastic)：
  - 户外/儿童场合
  - 不适合正式鸡尾酒（视觉质感差）

纸质杯：
  - 环保活动/外带
  - 不适合任何正式场合
```

> ⚠️ 适量饮酒，禁止酒驾。
