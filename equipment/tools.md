### 1.2 购买建议
```
⚠️ 不要买的：
  - 太小的量酒器（看不清刻度）
  - 玻璃摇酒壶（易碎、隔热差）

  - 不锈钢吧勺太轻（手感差）

✅ 值得多花一点钱的：
  - 量酒器（每天用，买手感好的）
  - 摇酒壶（决定你的效率和姿势）
```

## 二、进阶级工具（¥500-2000）

### 2.1 波士顿摇酒壶套装
```
推荐套装（波士顿壶 + 玻璃量杯 + Hawthorne 滤器）：
  - Boston Shaker Set（搜索"波士顿调酒套装"）
  - 价格：¥150-400

优点 vs 三段式：
  - 更快（职业调酒师首选）
  - 更大批量
  - 缺点：需要单独买滤器（套装通常含）

进阶动作：需要练习开壶（靠手腕巧劲）
```

### 2.2 量勺与滴管
```
鸡尾酒精密量具：
  - 吧勺 (Bar Spoon)：量约 5ml/勺
  - 滴管 (Eyedropper)：苦精精确控制
  - 小量勺套装：精确量取利口酒

价格：¥30-100
```

### 2.3 调酒杯 (Mixing Glass)
```
需求：需要一个玻璃或金属杯进行搅拌
规格：350-500ml 容量
推荐：厚底玻璃（耐撞）或不锈钢（耐用）
价格：¥50-200
```

### 2.4 柠檬/青柠榨汁器
```
手持榨汁器（ Citrus Juicer）：
  - 小型，价格 ¥30-80
  - 够用但不高效

手动压榨器（Mexican Elbow）：
  - 专业级，¥200-500
  - 效率高，出汁多
  - 海岛风格酒吧标配

电动榨汁器：
  - 不推荐（速度优先 vs 手压风味保留）
```

## 五、专业吧台布局

### 5.1 工作流布局原则
```
从左到右（调酒师视角）：

[冰桶] [冰夹] [冰镇酒杯]
   ↓
[摇酒壶/调酒杯] ← 基酒区 →
[量酒器] [吧勺] [滤器]
   ↓
[辅料区：果汁/糖浆/利口酒]
   ↓
[装饰区：柑橘/樱桃/香草]
   ↓
[出品台/客人面前]

关键原则：
  - 最常用的工具放在伸手可及处
  - 左撇子/右撇子都顺手
  - 不要跨越身体取东西
```

### 5.2 吧台配置参考
```
家庭吧台（5-10种酒）：
  - 基酒 5 瓶：金酒、伏特加、朗姆、龙舌兰、波本
  - 利口酒 3 瓶：君度、咖啡利口酒、甜味美思
  - 苦精 1-2 瓶：安高天娜、橙苦精
  - 糖浆 2-3 种：简单糖浆、生姜、蜂蜜
  - 柑橘：柠檬+青柠（必备）
  - 冰：提前制好

爱好吧台（20+种酒）：
  - 全部六大基酒
  - 每种基酒 1-2 款品牌差异
  - 完整利口酒收藏
  - 味美思全系列
  - 专业辅料（蛋清、椰子奶油等）
  - 苏打水/汤力水/姜汁啤酒常备
  - 制冰机或冰柜

专业吧台（50+种酒）：
  - 分门别类的完整收藏
  - 特殊利口酒
  - 异域风味
  - 客人定制能力
```

## 六、工具保养

```
金属工具：
  - 用后冲洗（尤其是滤器弹簧）
  - 每周深度清洗一次
  - 不锈钢可进洗碗机（手洗更佳）

木质捣棒：
  - 用后刷洗干净
  - 不要长时间泡水（会裂）
  - 定期用食品级矿物油保养

量酒器：
  - 检查刻度是否清晰
  - 内侧也要清洗
  - 不锈钢最耐用

滤冰器弹簧：
  - 每三个月检查弹性
  - 弹簧松弛 → 过滤效果差 → 换新
```
