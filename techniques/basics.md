# 基础技法 (Basic Techniques)

> 调酒的四大基本技法 + 辅助技法，附动作分解与常见错误。

---

## 一、摇和法 (Shaking)

### 1.1 工具认识
```
三段式摇酒壶 (Cobbler Shaker)：
  - 三件套：壶身 + 内盖 + 外盖
  - 最适合新手，密封性好
  - 缺点：壶身有接缝，清洗不便

波士顿摇酒壶 (Boston Shaker)：
  - 两件套：金属壶 + 玻璃杯
  - 专业首选，出品速度快
  - 需要单独购买滤冰器

锡制摇酒壶 (French Shaker)：
  - 两件套：金属壶身 + 内盖（无外盖）
  - 经典造型，吧台表演常用
  - 开盖需要用巧劲
```

### 1.2 动作分解（波士顿摇酒壶）
```
步骤一：准备
  - 摇酒壶分大壶（金属）和小壶（玻璃杯）
  - 大壶在下，小壶在上
  - 确保小壶嵌入大壶约 1cm

步骤二：加料
  - 先倒入液体原料
  - 再加冰（占壶容量 2/3，不是满的）
  - 冰太少 → 冷却不足，稀释不够
  - 冰太多 → 液体溅出，混合不均

步骤三：合壶
  - 45度倾斜合壶
  - 听到一声清脆"咔"即密封成功
  - 密封不严 → 摇的时候漏液

步骤四：摇动
  - 持壶姿势：食指抵住小壶顶端，拇指和中指夹住底部
  - 动作：上下大幅摇动
  - 节奏：每秒一次，共 10-15 秒
  - 听声音判断：叮叮当当 → 闷闷沙沙 = 到位

步骤五：开壶
  - 大壶在下，小壶在上
  - 用掌心或专用开壶器敲击接缝处
  - 一转即开，动作干净利落

步骤六：过滤出品
  - Hawthorne 滤冰器（有弹簧的那个）
  - 先放滤冰器，再倒酒
  - 如果有蛋清/果肉 → 双重过滤（滤冰器 + 细筛）
```

### 1.3 常见错误
| 错误 | 结果 | 修正 |
|------|------|------|
| 冰加太满 | 液体无法翻滚混合 | 2/3 冰量 |
| 摇太久 | 过度稀释，冰渣太多 | 10-15秒足够 |
| 没有密封就摇 | 漏液、飞溅 | 确认"咔"声再摇 |
| 只用蛮力 | 动作难看，效率低 | 用腰带动，不是只用手腕 |
| 原料顺序错 | 糖浆黏底 | 先液体后冰，糖浆先放 |

## 二、搅拌法 (Stirring)

### 2.1 动作分解
```
工具：调酒杯 (Mixing Glass) + 长吧勺

步骤一：准备
  - 调酒杯加冰（大冰块优先）
  - 冰块要干净，碎冰会影响透明度

步骤二：倒入原料
  - 量酒器精确量取
  - 沿杯壁倒入，减少气泡

步骤三：搅拌
  - 吧勺插入冰中，约 45 度角
  - 食指和中指夹住勺柄
  - 手腕发力，不是手臂
  - 沿杯壁画圆，速度均匀
  - 20-30 秒，感受杯壁温度变化
  - 手感变凉但不起雾 → 到位

步骤四：过滤
  - Julep 滤冰器（有弹簧的）放杯口
  - 轻轻倒入预冷酒杯
  - 不扰动底部冰碴

步骤五：装饰
  - 橙皮扭、柠檬皮、火烧橙皮等
  - 直接放入杯中或杯边
```

### 2.2 什么时候用搅拌？
- 全部是透明液体（酒+酒）
- 追求清澈透明视觉
- 需要温和稀释（烈性酒）
- **不适用于**：含果汁/奶/蛋清的配方

### 2.3 常见错误
| 错误 | 结果 | 修正 |
|------|------|------|
| 冰块太小/是碎冰 | 过度稀释 | 用 2×2cm 方冰 |
| 搅拌太快 | 产生气泡，混浊 | 稳定圆形，动作慢 |
| 时间不够 | 酒液不够凉，稀释不足 | 至少 20 秒 |
| 手握杯身 | 手温传导给酒 | 握杯底或用夹子 |

## 三、兑和法 (Building)

### 3.1 什么是兑和？
直接在饮用杯中依次倒入原料，不摇不搅拌。

### 3.2 兑和的操作要点
```
1. 杯具先行：杯子必须冰镇（提前放冷冻室）
2. 冰块先行：先加冰，再倒酒
3. 顺序讲究：
   - 先基酒后辅料
   - 碳酸类最后，沿杯壁注入（保护气泡）
   - 分层款则按密度从重到轻

4. 轻混合：Highball 类用吧勺提拉 1-2 次，不搅拌
5. 不过滤：不需要过滤，冰即饮品
```

### 3.3 兑和 vs 摇和
| 指标 | 兑和 | 摇和 |
|------|------|------|
| 稀释量 | 少（5-10%） | 多（25-30%） |
| 气泡保留 | 全部保留 | 部分丧失 |
| 口感 | 清脆、层次分明 | 圆润、融合 |
| 速度 | 快 | 慢 |
| 适合酒款 | Highball、米鸡尾 | Sour、含果汁/蛋 |

## 四、捣碎法 (Muddling)

### 4.1 工具：捣棒 (Muddler)
```
木制捣棒：最常见，质感好
金属捣棒：耐用，易清洁
塑料捣棒：轻便，但不推荐

选购标准：
- 长度：比摇酒壶高 5cm 以上
- 末端：平滑圆头（不要尖刺，会刺穿食材）
```

### 4.2 动作分解
```
1. 将原料放入杯底（水果切块/糖/香草）
2. 捣棒垂直插入
3. 手掌轻压，利用体重而非臂力
4. 动作：下压 → 轻微旋转 → 提起
5. 重复 3-5 次

力度原则：
  - 薄荷/草本：轻拍（提精油，不破叶）
  - 柑橘/水果：中力（出汁）
  - 糖块/姜：重力（溶解）
```

### 4.3 常见错误
| 错误 | 结果 | 修正 |
|------|------|------|
| 薄荷捣烂 | 苦味释放，整杯发苦 | 只轻拍，不捣碎 |
| 过分解压水果 | 出汁过多，过酸 | 适度，保留果肉纤维 |
| 旋转过度 | 冰碴太多 | 压完提起，少旋转 |

## 五、辅助技法

### 5.1 滤冰 (Straining)
```
Hawthorne 滤冰器（有弹簧）：
  - 通用型，带弹簧贴合杯口
  - 用于摇酒壶过滤

Julep 滤冰器（无弹簧）：
  - 用于调酒杯过滤
  - 无弹簧，直筒设计

细滤网：
  - 过滤蛋清泡沫/果肉
  - 配合以上两种使用（二次过滤）

双重过滤 = Hawthorne/Julep + 细滤网
```

### 5.2 装饰 (Garnishing)
```
橙皮扭 (Orange Twist)：
  1. 用削皮刀/削皮器取橙皮（只取皮，不带白瓤）
  2. 剪成约 5cm 长条
  3. 手指捏住两端，向外旋转
  4. 挂在杯边或投入杯中

柠檬皮扭：
  同上，换成柠檬

火焰橙皮 (Flamed Orange)：
  1. 橙皮取下后，皮面朝杯口
  2. 大拇指弹一下，让精油雾化
  3. 用打火机/火枪快速点燃皮边
  4. 火熄灭瞬间盖住杯口，让烟进杯
  5. 拿走橙皮，饮用

果皮的选择：
  - 橙皮：几乎所有深色鸡尾酒
  - 柠檬皮：酸酒类、透明酒款
  - 青柠皮：热带风格、龙舌兰酒
  - 柚子皮：日本风味
```

### 5.3 盐边与糖边 (Rimming)
```
盐边：
  1. 青柠/柠檬角切面沾湿杯边
  2. 杯口倒扣在盐盘上，转半圈
  3. 轻轻拍掉多余的盐

糖边：
  同上，换成糖粉
  可加食用色素做彩虹边

技巧：半边盐/半边糖 = Margarita 经典

进阶：烟熏盐、辣椒盐、咖喱盐、肉桂糖等
```

### 5.4 提拉法 (Rolling/Throwing)
```
用于：Highball 类，不想过度稀释但需要混合

动作：
  1. 两杯互倒
  2. 从高处（30cm）倒下，产生气泡
  3. 倒回原杯
  4. 重复 5-8 次

效果：混合 + 轻微充气 + 少量稀释
比搅拌更温和，比直接倒更混合
```

## 技法选择决策树

```
新酒需要什么？
  │
  ├─ 含果汁/奶/蛋清？
  │    └─ 是 → 摇和（必须乳化）
  │
  ├─ 全透明原料（纯酒+利口酒）？
  │    └─ 是 → 搅拌（保持清澈）
  │
  ├─ 需要视觉分层？
  │    └─ 是 → 兑和（按密度分层）
  │
  └─ 含需要提取的固体（薄荷/水果/糖块）？
       └─ 是 → 先捣碎/浸泡，再选择摇或搅拌
```

> 技法无高下，合适即最好。摇酒壶不代表低级，搅拌不代表高级。

> ⚠️ 适量饮酒，禁止酒驾。
