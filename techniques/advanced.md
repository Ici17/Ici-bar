# 进阶技法 (Advanced Techniques)

> 从干摇到分子调酒，从桶陈到脂肪洗 — 大师之路上的关键技术。

---

## 一、干摇与澄清

### 1.1 干摇 (Dry Shake)
```
定义：不含冰的摇和
目的：让蛋清/乳制品充分乳化，产生细腻泡沫

操作：
  1. 原料（含蛋清）放入摇酒壶
  2. 不加冰
  3. 用力摇 15-20 秒（听声音：哗哗的水声→变稠）
  4. 加冰，再摇 15 秒（降温）
  5. 双重过滤出品

关键点：
  - 力气要大，干摇是纯粹的物理乳化
  - 没有冰的帮助，必须靠速度和次数
  - 蛋清充分乳化的标志：倒出时有明显泡沫挂杯
```

### 1.2 反向干摇 (Reverse Dry Shake)
```
定义：先加冰摇，再去掉冰干摇
优点：比传统干摇泡沫更细腻、持久

操作：
  1. 正常加冰摇 15 秒
  2. 滤出冰块（酒液留在壶内）
  3. 无冰再摇 15 秒（打泡沫）
  4. 直接倒出（不用滤器，泡沫一起出）

适合：Pisco Sour、Ramos Gin Fizz
```

### 1.3 牛奶澄清 (Milk Clarification)
```
原理：蛋白质遇酸凝固，吸附杂质，澄清液体

操作（以 Milk Punch 为例）：
  1. 波本 + 柠檬汁 + 糖浆 + 牛奶 混合
  2. 轻轻搅拌，静置 30 分钟（蛋白质絮凝）
  3. 细滤纸（咖啡滤纸）过滤
  4. 滤出乳白色透明液体

结果：丝绒口感但视觉清澈的神奇效果
代表：Milk Punch、Boston Clarity

技巧：
  - 牛奶用全脂，效果最好
  - 酸要分次加入，边加边观察絮凝
  - 过滤极慢（可能需要 1 小时），提前准备
  - 一次可做大批量，冷藏可存 3 天
```

## 二、浸泡与洗酒

### 2.1 油脂浸泡 (Fat Washing)
```
原理：非极性溶剂（酒精）溶解脂溶性风味物质

操作步骤：
  1. 选脂肪（黄油/培根油/椰子油/芝麻油）
  2. 加热至 50-60℃，脂肪融化
  3. 倒入基酒，搅拌均匀
  4. 常温静置 2-4 小时（或冰箱过夜）
  5. 冷冻过夜（脂肪上浮凝固）
  6. 用咖啡滤纸过滤（脂肪被去除）

配方计算：
  - 脂肪量：基酒的 5-15%
  - 浸泡时间越长，脂肪风味越重
  - 可做小批量测试最佳比例

经典应用：
  - 培根波本 → Bacon Old Fashioned
  - 棕黄油波本 → Brown Butter Sour
  - 椰子朗姆 → Coconut Rum Sour
  - 芝麻油金酒 → Sesame Gin & Tonic
```

### 2.2 快速浸泡 (Quick Infusion / Turbo Infusion)
```
使用工具：Whip Cream Chargers（N2O 气弹）或 sous-vide

气弹快速浸泡：
  1. 原料（香草/香料/水果）放入气弹罐
  2. 倒入基酒
  3. 装上气弹，加压
  4. 摇 30 秒，放气
  5. 过滤

Sous-vide 浸泡（更均匀）：
  1. 原料+基酒入真空袋
  2. 55℃ 水浴 1-2 小时
  3. 冷却，过滤

适用：香草精、辣椒、水果、茶
```

### 2.3 冷萃 (Cold Brew / Cold Extraction)
```
适合：茶、咖啡、香草（怕热分解的原料）

方法一：冷萃壶
  1. 原料放入冷萃壶
  2. 加基酒，盖好
  3. 冰箱冷藏 12-24 小时
  4. 过滤

方法二：搅拌法
  1. 冰块+原料+基酒入搅拌机
  2. 搅拌 2 分钟
  3. 过滤

代表：冷萃咖啡马天尼、冷萃伯爵茶金酒
```

## 三、桶陈 (Barrel Aging)

### 3.1 家庭桶陈方案
```
工具：小橡木桶（5-25升）或橡木片

小橡木桶（Amazon有售，约¥200-500）：
  优点：真实桶陈效果，可重复使用
  缺点：占用空间，需要保养

橡木片（更灵活）：
  优点：成本低，用量可控
  缺点：不是真正的桶陈

方法：
  1. 新桶：用热水浸泡 24 小时（让木头膨胀密封）
  2. 老桶：冲洗后晾干
  3. 倒入预调鸡尾酒（酒液占桶 80-90%）
  4. 记录起始日期和目标时间
  5. 每 3 天取样品尝
  6. 达到目标风味后，瓶装冷藏
```

### 3.2 推荐陈化时间
| 酒款类型 | 桶陈时间 | 判断标准 |
|---------|---------|---------|
| Old Fashioned | 1-2 周 | 苦精风味融入，木质香 |
| Negroni | 3-4 周 | 三者融合，略微收敛 |
| Manhattan | 1-2 周 | 甜味美思与威士忌融合 |
| Daiquiri 风格 | 1-2 周 | 酸甜更圆润 |
| 单一基酒 | 4-8 周 | 获得橡木桶风味 |

### 3.3 桶陈注意事项
```
1. 温度越高，陈化越快（但不要超过室温25℃）
2. 酒精度数越高，陈化越快
3. 桶越小，陈化越快（5升桶快于25升桶）
4. 每批陈化后，桶会"记忆"之前的风味，需要清桶
5. 清桶方法：热水冲洗 + 晾干 + 必要时用威士忌涮一遍
6. 陈化过头比陈化不足更难补救
```

## 四、洗杯与油封

### 4.1 苦艾酒洗杯 (Absinthe Rinse)
```
代表：Sazerac

操作：
  1. 少量苦艾酒（约 5ml）倒入古典杯
  2. 旋转，让酒液挂满整个杯壁
  3. 倒掉多余苦艾酒
  4. 杯中只留下附着在杯壁的酒液薄膜

效果：苦艾的茴香和大茴香风味弥漫整个口腔

注意：苦艾酒应选正经品牌（不是那种有摇头丸嫌疑的"苦艾酒"）
推荐：Lucid（美国合法进口）、Pernod（法国经典）
```

### 4.2 橄榄汁洗杯 (Olive Juice Rinse)
```
代表：Dirty Martini

操作：
  1. 橄榄汁（从橄榄罐里取，或手动压橄榄）
  2. 约 5ml 入杯
  3. 旋转挂杯
  4. 倒掉多余（或留一点在杯底）

效果：微咸、鲜味，提升马天尼的复杂度

替代：Gibson 用 cocktail onion 的汁
```

## 五、泡沫与烟熏

### 5.1 虹吸瓶泡沫 (Siphon Espuma)
```
工具：ISI 虹吸瓶 + 气弹（N2O）

操作：
  1. 液体原料入虹吸瓶
  2. 拧紧盖子
  3. 装入一颗 N2O 气弹
  4. 摇匀，倒置
  5. 按压即出泡沫

配方：酸性柚子foam
  - 柚子汁 200ml + 糖浆 30ml + 吉利丁 2g
  - 加热溶解吉利丁 → 入虹吸瓶 → 冷却 → 气弹

适合：顶层加泡沫的鸡尾酒（不影响下层味道）
```

### 5.2 蛋清泡沫 (Foam)
```
无气弹方法（用打蛋器）：
  1. 蛋清 + 少许酸性液体（稳定泡沫）
  2. 电动打蛋器打至湿性发泡（6-7分发）
  3. 用勺子轻轻放在饮品顶层
  4. 或用裱花袋挤在顶层

酸性稳定剂：几滴柠檬汁或酒石酸
```

### 5.3 冷烟熏 (Cold Smoking)
```
工具：烟熏枪 + 木屑

操作：
  1. 调好的酒倒入杯中
  2. 玻璃钟罩（或任何能罩住杯子的东西）
  3. 点燃烟熏枪，注入烟雾
  4. 盖住，等待 30-60 秒
  5. 揭盖，烟雾散去，只留风味

木屑选择：
  - 苹果木：果香、温和
  - 樱桃木：甜香
  - 橡木：经典
  - 泥煤(Islay)：烟熏、强烈
  - 山核桃木：BBQ 风味

适用酒款：Old Fashioned、Whiskey Sour、马天尼

警告：不要在密闭空间大量使用
```

## 六、分子调酒入门

### 6.1 球化技术 (Spherification)
```
材料：海藻酸钠（Sodium Alginate）+ 氯化钙（Calcium Chloride）

基础操作（以果汁球为例）：
  1. 果汁 + 海藻酸钠（1g/100ml）混合 → 冷藏消泡
  2. 氯化钙水溶液（1g/100ml）入另一碗
  3. 用勺子/滴管取果汁混合物
  4. 滴入氯化钙溶液中
  5. 30 秒内，液滴表面形成凝胶球
  6. 捞出，用清水冲洗
  7. 冷藏保存（24小时内使用）

应用：Mango Caviar（芒果鱼子酱）、Yuzu Pearls

关键：液体必须是含钙的（如果汁），不加额外钙
替代：反向球化（海藻酸钠放钙液中，外层液体放海藻酸钠液中）
```

### 6.2 旋转蒸发 (Rotavap)
```
高端设备，酒吧少见
原理：减压低温蒸发，获取纯净风味浓缩物

家庭替代：蒸馏提取（更复杂，不建议新手）
```

## 七、自制糖浆与调味液

### 7.1 基础糖浆系列
```
简单糖浆 (Simple Syrup)：
  1:1 — 水:糖，加热溶解，冷藏可用 2 周
  2:1 (Rich) — 更浓稠，甜度更高，保存更久

蜂蜜糖浆：
  蜂蜜 + 水（1:1），加热溶解
  可加：生姜、辣椒、迷迭香等

香草糖浆：
  1:1 糖浆 + 香草荚（剖开），浸泡 24 小时

香料糖浆：
  八角/肉桂棒/丁香 + 1:1 糖浆，浸泡 1 周

智利糖浆 (Chile Syrup)：
  墨西哥辣椒切片 + 1:1 糖浆，浸泡 3 天
```

### 7.2 酸修饰液 (Acid Adjusted Juice)
```
目的：替代鲜榨果汁，保存更久，风味更稳定

柠檬酸修饰液：
  - 柠檬汁 100ml
  - 柠檬酸粉 3g
  - 酒石酸粉 1g
  - 混合溶解，冷藏可存 5 天

效果：比鲜榨更酸、更稳定，但永远不如新鲜好喝
建议：只在无法获得新鲜果汁时使用
```

## 技法安全警示

| 技法 | 潜在风险 | 安全措施 |
|------|---------|---------|
| 液氮 | 严重冻伤 | 戴手套，禁止入口温度直接接触 |
| 烟熏 | 吸入风险 | 通风环境，适量使用 |
| 脂肪洗冷冻 | 凝固分离不彻底 | 充分过滤，不要省滤纸 |
| 桶陈 | 过度氧化 | 定期品尝，记录时间 |
| 球化 | 误食钙粉 | 确保钙溶液彻底冲洗干净 |

> ⚠️ 适量饮酒，禁止酒驾。进阶技法请在安全环境下操作。
