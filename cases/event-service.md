# 实战案例：活动定制服务 (Event Service)

> 婚礼、派对、公司活动、生日会的鸡尾酒服务全流程。

---

## 一、活动类型与服务模式

### 1.1 常见活动类型
| 活动 | 特点 | 服务模式 | 人均消费 |
|------|------|---------|---------|
| 婚礼/婚宴 | 正式、长时间、混合客群 | 流动吧台/固定吧台 | ¥150-300/人 |
| 生日派对 | 轻松、高峰集中、气氛导向 | 移动吧台/围桌服务 | ¥100-200/人 |
| 公司活动 | 正式/半正式、人数多、时段短 | 固定高脚吧台 | ¥120-250/人 |
| 品牌发布会 | 精致、人数少、形象要求高 | 站立式展台 | ¥200-500/人 |
| 私人派对 | 风格化、高度定制 | 依客户需求 | ¥150-400/人 |
| 音乐节/市集 | 高流量、速度优先 | 快闪吧台 | ¥80-150/人 |

## 二、服务模式详解

### 2.1 流动鸡尾酒服务 (Mobile Bar)
```
标准配置：
  - 1 名调酒师 + 1 名助手
  - 可服务 50-100 人/3小时
  - 携带工具箱：摇酒壶、量酒器、冰桶、基础装饰

套餐内容（举例）：
  - 基础鸡尾酒 5 款 ¥5,000
  - 进阶鸡尾酒 8 款 ¥8,000
  - 含所有原料/耗材/运输
```

### 2.2 完整吧台 (Full Bar)
```
配置：
  - 2-3 名调酒师
  - 吧台设备租赁
  - 可服务 100-200 人/4小时

适合：婚礼、公司年会
```

## 三、婚礼鸡尾酒服务案例

### 案例背景
```
新人：李先生 & 王小姐
婚礼地点：上海市郊花园婚礼场地
时间段：下午 4:00-6:00（仪式后鸡尾酒会）
宾客：约 80 人
预算：¥15,000
季节：秋季
主题："金色秋天，新旅程"
```

### 3.1 酒单设计
```
设计原则：
  - 男女皆宜（不要过于烈性）
  - 有故事感（配合婚礼主题）
  - 含无酒精选项
  - 颜色与婚礼配色协调（秋天金色/橙色）

酒单（6款）：

【金色系列】
1. Golden Hour（黄金时刻）
   波本 + 苹果汁 + 枫糖浆 + 肉桂
   ¥88/杯

2. Autumn Negroni
   金酒 + 金巴利 + 甜味美思
   ¥78/杯

【女士友好】
3. Rose Garden
   金酒 + 玫瑰糖浆 + 西柚汁 + 蛋清
   ¥85/杯

4. Lychee Sparkling
   伏特加 + 荔枝利口酒 + 气泡水
   ¥75/杯

【男士选择】
5. Smoky Old Fashioned
   波本 + 安高天娜苦精 + 墨西哥橙皮
   ¥82/杯

【无酒精】
6. Autumn Shrub
   苹果醋饮 + 枫糖浆 + 苏打水
   ¥35/杯
```

### 3.2 服务流程
```
时间线：
  14:00 - 设备到场，搭建吧台
  15:00 - 冰块/原料就位，杯具冰镇
  15:30 - 最后准备，调酒师备装
  16:00 - 仪式结束后，迎宾开始
  16:00-16:30 - 第一轮（仪式高峰，快速出品）
  16:30-17:30 - 第二轮（轻松节奏）
  17:30-18:00 - 收尾，清理

出杯速度：
  - 峰值：60杯/30分钟 = 2杯/分钟
  - 准备：3名调酒师 + 1助手
  - 每款预调 10 杯量基酒混合物（加快出品）
```

### 3.3 备料计算
```
80人 × 2杯 = 160杯（含无酒精）
加 20% 保险 = 192杯

每款（约 27 杯）原料：
  Golden Hour：
    波本 27 × 45ml = 1,215ml ≈ 3 瓶
    苹果汁 27 × 30ml = 810ml ≈ 1 瓶
    枫糖浆 27 × 15ml = 405ml ≈ 1 瓶

  Autumn Negroni：
    金酒 27 × 30ml = 810ml ≈ 2 瓶
    金巴利 27 × 30ml = 810ml ≈ 2 瓶
    甜味美思 27 × 30ml = 810ml ≈ 2 瓶

总原料成本估算：¥3,000-4,000（20-25%）
```

## 四、公司活动案例

### 案例背景
```

活动：年终派对
地点：公司大楼天台花园
时间段：19:00-22:00（3小时）
```

```
1. Classic G&T（改良版）

   ¥58/杯

   伏特加 + 姜啤 + 青柠
   ¥58/杯

3. Aperol Spritz
   Aperol + 普罗塞克 + 苏打水
   ¥68/杯

   Seedlip + 汤力水 + 黄瓜
   ¥38/杯
```

```
吧台配置：3个服务站，每站 1 调酒师

每个站点：
  - 1个冰桶（预制混合物）
  - 2个分配瓶（预调基酒混合物）
  - 1个苏打水分配瓶

  - 装饰（青柠角/黄瓜条）

出杯目标：60杯/站点/小时 = 180杯/小时
总出杯：540杯/3小时（超过200人需求）
```

```
出杯速度目标：30秒/杯（高峰）
```

## 六、活动服务注意事项

### 6.1 常见坑
```
❌ 预估人数不准 → 原料浪费或不够
  → 方案：合同注明"按实际人数结算"

❌ 天气影响（户外）→ 原料/设备损坏
  → 方案：提前查看天气预报，备防水布

❌ 客人喝醉 → 安全事故
  → 方案：设置无酒精选项，监控客人状态

❌ 电源/设备问题 → 无法正常工作
  → 方案：自带延长线/发电机

❌ 清理不彻底 → 被扣押金
  → 方案：合同注明清理责任，留够时间
```

### 6.2 合同必备条款
```
1. 服务时间（精确到分钟）
2. 服务人数（预估+上限）
3. 酒单确认（提前 1 周定稿）
4. 价格锁定（不因原料波动调价）
5. 超时费（¥/小时）
6. 取消政策（提前 X 天全额退款）
7. 损坏赔偿（设备/场地）
8. 保险（公共责任险）
9. 食材安全（食品安全许可证）
10. 清洁责任（双方各自负责什么）
```

## 七、定价参考

```
活动鸡尾酒服务报价逻辑：

基础报价 = (原料成本 × 3) + 服务费

示例（婚礼鸡尾酒会）：
  原料成本：¥4,000
  加工费（3人团队 6小时）：¥3,000
  设备租赁：¥1,500
  交通/物流：¥800
  利润：¥2,700
  合计报价：¥12,000

折扣空间：10-15%（大客户/介绍）
不打折项：原料成本（品质不能降）
```

> ⚠️ 适量饮酒，禁止酒驾。活动服务需持有相关营业执照。
