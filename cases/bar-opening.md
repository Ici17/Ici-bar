# 实战案例：精品酒吧开业 (Bar Opening)

> 从 0 到 1 开一家有灵魂的精品鸡尾酒吧。覆盖定位、选品、酒单设计、备料、成本、运营。

---

## 案例背景

```
项目名称：The Compass Bar（假设）
地点：上海法租界（年轻文艺社区）
面积：约 80 平方米，30 个座位
预算：启动资金 80 万
目标客群：25-40 岁，鸡尾酒爱好者，周末社交
开业时间：2025 年春季
```

---

## 一、定位与概念

### 1.1 定位三问
```
问自己：

1. 我的酒吧解决什么需求？
   答：社区需要一个"安静但不沉闷"的精品酒吧，
       不是吵杂的夜店，而是可以认真喝酒的地方。

2. 我的竞争对手是谁？我和它们有什么不同？
   差异化：
   - 传统酒吧：大众酒单，无特色
   - 网红酒吧：颜值高但酒一般
   - 我们的不同：酒单有故事，每款酒可溯源，定期更换当季主题

3. 我的客人喝完酒后想说什么？
   "这杯酒很有意思，我想知道更多关于它的故事。"
```

### 1.2 概念文案
```
名称：The Compass Bar
标语："每杯酒都是一个方向"

概念：
- 酒单按"方位"分类（北/南/东/西）
- 每款酒背后有地理/文化故事
- 强调"可以认真喝一杯"的空间氛围

```

## 二、酒单设计

### 2.1 酒单结构（15-18款）
```
第一部分：指南针经典（6款）
  - 涵盖六大基酒，每款一种经典
  - 安全牌，让新客人有选择

第二部分：本地故事（5款）
  - 原创配方，灵感来自上海/中国文化
  - 例：桂花马天尼、酱油古典、白酒酸酒

第三部分：世界旅行（4款）
  - 以目的地为灵感的鸡尾酒
  - 例：古巴哈瓦那（莫吉托变体）、日本京都（抹茶马天尼）

第四部分：无/低酒精（3款）
  - Seedlip / 醋饮 / 苏打水
  - 照顾不喝酒的客人

```

## 四、开吧 SOP（Standard Operating Procedure）

### 4.1 开吧前 30 分钟
```
□ 检查当日库存（昨晚退库+今早补货）
□ 切柠檬/青柠（按预估量 × 1.2 系数）
□ 预制糖浆检查（是否在 3 天内）
□ 制作当日装饰（柑橘扭/薄荷/樱桃）
□ 冰块检查（昨晚补满了吗？）
□ 设备检查（搅拌机/榨汁器/热水）
□ 抹布/台面清洁
□ 杯具检查（有破损/水渍？）
□ 音乐播放列表准备
□ 灯光/氛围灯调试
□ 价格牌/酒单更新
□ 员工早会（今日特调/特殊客人）
```

### 4.2 营业中
```
□ 每 2 小时补一次冰
□ 鲜榨果汁（每 3 小时检查鲜度）
□ 糖浆使用前摇匀
□ 每 30 分钟整理吧台
□ 记录每款酒的销量（观察偏好）
□ 客人反馈记录
□ 及时补充瓶装基酒
```

### 4.3 收吧 SOP
```
□ 最后点单广播（提前 15 分钟）
□ 冰块处理（剩余退回冰柜/处理）
□ 剩余果汁处理（超过 4 小时？倒掉）
□ 糖浆密封冷藏
□ 瓶装基酒归位
□ 清洁所有工具
□ 清洁台面/地面
□ 垃圾处理
□ 记录今日销量和明日备忘
□ 关闭非必要电源
□ 上锁
```

## 五、成本控制表

### 5.1 月度经营指标
```
目标数据：
  日均杯数：20 杯
  月营收目标：¥45,000
  月运营成本：¥25,000（含租金）
  月利润目标：¥20,000

成本结构（预估）：
  租金：¥12,000
  人工（1名全职+兼职）：¥8,000
  原料：¥8,000
  水电杂费：¥2,000
  耗材：¥1,000
  总成本：¥31,000

盈亏平衡点：
  月固定成本 ¥23,000（不含原料）
  盈亏平衡 = 23,000 / 0.75 = ¥30,667/月
  日均杯数 = 30,667 / ¥75 / 30天 = 13.6 杯/天
```

## 六、运营关键指标

```
每日追踪：
  - 每款酒的销售杯数
  - 每款酒的原料成本
  - 客人平均停留时间
  - 高峰时段（优化人力）

每周分析：
  - 最畅销的 5 款 / 最滞销的 3 款
  - 滞销酒：下架 or 改配方？
  - 客人反馈 TOP 3

每月复盘：
  - 原料成本率（目标 < 25%）
  - 毛利率
  - 客人复购率
  - 员工表现评估
```

## 七、常见错误清单

```
❌ 开吧常见错误：
  1. 酒单太多款（新手先做 10-12 款）
  2. 鲜榨果汁一次切太多（24小时内用完）
  3. 不记录每杯酒的配方（无法复盘）
  4. 忽略无酒精选项（流失 20% 客人）
  5. 定价凭感觉（必须基于成本）
  6. 忽视库存管理（过期浪费是隐形杀手）
  7. 备料不足（高峰期断货 = 客人流失）
```
