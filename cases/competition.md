# 实战案例：调酒比赛准备 (Competition Preparation)

> 从地方赛到全国赛，如何系统地准备一场调酒比赛。

---

## 一、比赛类型速查

| 赛事 | 主办 | 形式 | 特点 |
|------|------|------|------|
| 中国国际调酒大师赛 | 中国酒业协会 | 地区赛+总决赛 | 国内最权威 |
| 百加得传世鸡尾酒大赛 | 帝亚吉欧 | 创意调酒 | 全球规模最大 |
| 芝华士鸡尾酒大师赛 | 保乐力加 | 大师赛制 | 威士忌专项 |
| 绝对伏特加调酒大赛 | 绝对 | 创意+视觉 | 视觉创意导向 |
| IBA World Cocktail Championship | IBA | 官方赛事 | 规则最严格 |
| 地方精酿酒吧联赛 | 各地自行组织 | 不定 | 门槛低 |

## 二、参赛准备时间线

### 2.1 六个月参赛计划
```
【第1-2个月：定位】
□ 选择参赛类别（经典/创意/大师/花式）
□ 确定基酒品牌（通常由赞助商指定）
□ 研究历届获奖配方（找规律）
□ 找到自己的调酒哲学

【第3-4个月：创作】
□ 方向头脑风暴（至少 20 个想法）
□ 淘汰到 5 个方向
□ 每个方向做 3 款原型
□ 邀请同行/导师盲测反馈
□ 确定最终 1 款配方

【第5个月：打磨】
□ 配方精修（10-20轮迭代）
□ 命名与故事包装
□ 出品设计（杯具/装饰/温度）
□ 现场演练（计时）
□ 话术准备（为什么选这杯？）

【第6个月：冲刺】
□ 每周 2-3 次完整演练
□ 模拟压力环境（嘈杂/时间限制）
□ 服装/形象准备
□ 了解比赛规则细则
□ 提前踩点比赛场地
□ 物资清单准备
```

## 三、配方设计方法论

### 3.1 配方创作流程
```
第一步：灵感触发
  来源：旅行 / 音乐 / 书籍 / 个人经历 / 季节 / 味道
  例："小时候外婆做的桂花糕"

第二步：核心概念
  形式化你的灵感
  例："童年记忆中桂花与秋天的联系"

第三步：风味地图
  核心味道 → 支撑味道 → 平衡味道
  桂花(核心) + 蜂蜜(支撑) + 柠檬(平衡)

第四步：技术实现
  选什么基酒？什么技法？
  金酒（中性，承载桂花）+ 搅拌法（保留花香）

第五步：迭代测试
  记录每一次调整：
  v1: 50ml Gin + 15ml 桂花酿 + 10ml 柠檬
      → 桂花味太淡
  v2: 50ml Gin + 20ml 桂花酿 + 15ml 柠檬
      → 甜度合适，但酸不够
  v3: 50ml Gin + 20ml 桂花酿 + 18ml 柠檬
      → OK，但桂花可以更明显
  v4: 55ml Gin + 22ml 桂花酿 + 16ml 柠檬 + 1dash Orange Bitters
      → 完成
```

### 3.2 命名策略
```
好名字的标准：
  1. 与配方有关联（不是空穴来风）
  2. 容易记住（短，有节奏感）
  3. 有故事暗示（名字即故事）

好名字示例：
  ✅ "The Wandering Botanist"（游荡的植物学家）
     暗合金酒中的植物成分，与旅行主题呼应
  
  ✅ "Autumn in Guangzhou"
     直接点明地域和季节

坏名字示例：
  ❌ "Cocktail #7"（无意义）
  ❌ "Delicious Yum"（太口水）
  ❌ 10个字以上的名字（记不住）

```

## 四、评分维度分析

### 4.1 IBA 国际评分标准
| 维度 | 分值 | 考核内容 |
|------|------|---------|
| 外观 | 20 | 视觉呈现、杯具选择、装饰 |
| 香气 | 20 | 香气复杂度、香气与风味一致 |
| 味道 | 30 | 平衡性、复杂度、余韵 |
| 技术 | 20 | 动作流畅、效率、时间控制 |
| 呈现 | 10 | 话术、故事性、整体印象 |

### 4.2 突破关键
```
最容易失分的点：
  ❌ 味道失衡（最常见失败原因）
    → 解法：至少 20 次盲测迭代

  ❌ 技术不熟练（紧张导致动作卡顿）
    → 解法：200 次以上的完整演练

  ❌ 故事讲不好（紧张时忘词）
    → 解法：写下 200 字以内的话术，背到自然

  ❌ 超时（专业赛常见）
    → 解法：严格计时，目标时间 = 限制时间 - 30秒

最容易被忽视的得分点：
  ✅ 余韵（30% 分数在这里）
    → 不要只关注入口，要关注回味
  
  ✅ 话术（10% 但很少人认真准备）
    → 1 分钟内说清楚"为什么是这杯酒"
```

## 五、话术模板

### 5.1 自我介绍 + 配方阐述（60秒）
```
模板：
"我叫 [名字]，来自 [城市/酒吧名]。
这杯酒叫 [酒名]。

它的灵感来自 [你的故事/概念]。

在风味上，我想呈现 [核心味道描述]，通过 [技术/原料] 实现。
[加入一个技术亮点说明，比如'我用自制的桂花糖浆替代商业版，因为它更纯净'。

这杯酒，我想表达的是 [一句话总结，20字以内]。

谢谢。"

参考示例：
"我叫李明，来自上海 The Compass Bar。
这杯酒叫'黄金时刻'。

它的灵感来自我父母结婚那天的秋天——金色的阳光、苹果派的香气。
在风味上，我想用波本的温暖承载苹果的甜美，用肉桂呼应那个年代的质感。

技术上，我用肉桂糖浆（自制，浸泡72小时）和2 dashes的安高天娜平衡甜度。
这杯酒，我想表达的是：每一段旅程，都有一个值得庆祝的开始。谢谢。"
```

### 5.2 评委 Q&A 准备
```
常见问题清单：

Q: 为什么选这个基酒？
A: [基酒特性] + [与配方主题的关联]

Q: 你的平衡逻辑是什么？
A: [甜/酸/苦/醇的具体数值] + [为什么这样设计]

Q: 这款酒的独特卖点是什么？
A: [一句话] + [技术支持]

Q: 如果再做一次，你会改什么？
A: [诚实承认一个弱点] + [改进方向]
  （这说明你有反思能力）

Q: 你觉得这量产的难点在哪？
A: [诚实分析] + [解决方案]
```

## 六、实战演练清单

### 6.1 赛前一周
```
□ 配方最终锁定（不要临时改）
□ 所有原料采购完毕
□ 工具检查（备用滤器/量酒器）
□ 服装熨烫
□ 交通/住宿确认
□ 比赛规则再读一遍（不要因规则失分）

□ 早睡！
□ 不要喝太多（保持味觉敏感）
```

### 6.2 比赛当天
```
□ 提前 2 小时起床
□ 吃清淡早餐（保护味觉）
□ 提前 1 小时到场踩点
□ 熟悉设备位置
□ 检查原料（比赛提供的可能品质一般，协商自带关键原料）
□ 深呼吸

比赛流程：
  □ 调酒前：快速确认所有原料在位
  □ 调酒中：专注于动作，不要看观众
  □ 调酒后：立即整理台面（也是评分项）
  □ 呈现：自信但不傲慢
  □ Q&A：诚实、有深度
```

## 八、比赛心态

```
⚠️ 比赛真相：
  - 80% 的输赢在配方设计阶段就决定了
  - 现场表现是放大部分，不是决定部分
  - 评委喜欢有故事的酒，不是炫技的酒

心态管理：
  ① 不要想着"赢"，想着"呈现最好的自己"
  ② 紧张是正常的，适度紧张让你更专注
  ③ 评委问刁钻问题是好事（说明他们在认真对待你）
  ④ 输赢之外：认识同行、学到新东西、被人记住

"最好的比赛表现，是让评委在喝完那杯酒之后，想再喝一口。"
— Dale DeGroff
```

> ⚠️ 适量饮酒，禁止酒驾。比赛需年满18周岁。
