# 装饰艺术 (Garnishes)

> 装饰是鸡尾酒的"封面"，也是香气和风味的入口。
> 这一模块覆盖从最简单的橙皮扭到复杂的糖艺装饰。

---

## 一、装饰的功能

```
装饰不是可有可无：

1. 视觉价值 — 颜值即正义，好看的酒先被选择
2. 香气来源 — 橙皮扭释放精油，这是真实的风味
3. 风味暗示 — 樱桃暗示甜，薄荷暗示清新
4. 文化符号 — 橄榄代表马天尼，洋葱代表吉布森
5. 互动入口 — 客人看到装饰，会自然地闻/摸/扭

⚠️ 无意义的装饰 > 有意义的装饰：
   放一片不搭的柠檬 = 噪音
   放一片强化故事的柠檬 = 信号
```

## 二、柑橘皮装饰

### 2.1 橙皮扭 (Orange Twist)
```
工具：削皮刀/蔬菜削皮器

步骤：
  1. 削皮器取橙皮（只取黄色部分，不要白色海绵层）
  2. 剪成 5-7cm 长条
  3. 手指捏住两端，向外旋转
  4. 挂杯边或投入杯中

手法要点：
  - 旋转让皮自然弯曲
  - 不要折断
  - 扭曲的形状 = 更宽的精油释放面积
```

### 2.2 火焰橙皮 (Flamed Orange)
```
步骤：
  1. 取橙皮（保留完整圆形，或长条）
  2. 皮面朝杯口上方
  3. 大拇指按住皮的一端，向外弯曲
  4. 快速用打火机/火枪点燃弯曲部分
  5. 火苗窜起时，立即用杯盖住（或直接翻转盖住）
  6. 火熄灭瞬间，精油雾化附在杯内
  7. 拿走橙皮，饮用

⚠️ 安全：点燃时间极短（1秒），不要烧手
```

### 2.3 柠檬/青柠扭
```
方法同上，区别：
  - 柠檬皮：更适合透明酒款、酸酒类
  - 青柠皮：热带风格、龙舌兰酒
  - 柚子皮：日本风味
```

### 2.4 橙皮轮 (Orange Wheel)
```
做法：
  1. 橙子横切成 5mm 圆片
  2. 切透（不是半圆）
  3. 杯边切一口子
  4. 套在杯边

用法：长饮类（Mojito、Gin & Tonic）
```

### 2.5 柠檬/青柠盐边
```
步骤：
  1. 柠檬/青柠角切面湿润
  2. 杯口倒扣在盐盘上
  3. 轻轻转动半圈
  4. 轻敲去除多余盐粒

变体：半边盐/半边糖（Margarita 经典）
```

## 三、香草与植物装饰

### 3.1 薄荷叶
```
放置技巧（Mojito / Mint Julep）：
  - 不只是"插在上面"
  - 轻拍醒香（手心拍一下再放）
  - 大量使用（5-10片）= 更强香气
  - 薄荷叶放入后轻提一下，让香气释放

保存：湿纸巾包裹，冷藏可存 2-3 天
```

### 3.2 迷迭香 / 百里香
```
用途：威士忌、古典鸡尾酒
处理：
  - 新鲜迷迭香用火苗轻燎一下（激活精油）
  - 百里香可直接使用
  - 两种都可以用糖浆固定在杯边

注意：迷迭香香气很强，用量要克制
```

### 3.3 罗勒叶
```
用途：适合金酒、Gin & Tonic
处理：
  - 轻拍醒香（同薄荷）
  - 整叶放，或撕碎放顶层
  - 配对：Gin + Strawberry + Basil

保存：同薄荷
```

### 3.4 新鲜香料
| 香料 | 配对 | 用法 |
|------|------|------|
| 肉桂棒 | 苹果、白兰地、威士忌 | 搅拌棒或杯边装饰 |
| 丁香 | 热饮酒、苹果酒 | 插在橙片上 |
| 八角 | 中国风味白酒 | 浸泡后取出 |
| 香草荚 | 巧克力、咖啡鸡尾酒 | 刮擦增加香气 |
| 花椒粒 | 中国白酒鸡尾酒 | 轻拍增加麻感 |

## 四、水果装饰

### 4.1 樱桃
```
类型：
  - 鸡尾酒樱桃（马拉斯奇诺，糖渍）
    推荐：Luxardo（最经典，深红色）
  - 鸡尾酒樱桃（普通染色）
    不推荐，颜色不正

放置：牙签穿串，或直接投入杯中
注意：不要用太甜的（掩盖酒味）
```

### 4.2 菠萝
```
菠萝角：
  - 切下一片，沿边缘卷起
  - 挂在杯边

菠萝叶：
  - 只取顶部绿叶部分
  - 插入杯中（热带风格）

菠萝块：
  - 立方体，用牙签穿
  - 直接放入杯中
```

### 4.3 浆果类
| 浆果 | 配对 | 用法 |
|------|------|------|
| 蓝莓 | 金酒、蓝柑橘 | 捣碎或整颗 |
| 草莓 | 朗姆、金酒 | 切片或整颗 |
| 覆盆子 | 黑莓利口酒、金酒 | 可捣碎增加颜色 |
| 黑莓 | 波本、苏格兰 | 整颗或捣碎 |

## 五、糖艺装饰

### 5.1 糖边 (Sugar Rim)
```
步骤：
  1. 糖盘（纯白糖/彩糖/肉桂糖）
  2. 柠檬/青柠角湿杯口
  3. 倒扣在糖盘上，转半圈
  4. 轻敲去多余糖粒

变体糖：
  - 肉桂糖：白糖+肉桂粉(3:1)
  - 辣椒糖：白糖+辣椒粉(极少量)
  - 彩虹糖：不同颜色层叠
```

### 5.2 糖片装饰
```
自制糖片：
  1. 糖浆（浓稠，2:1）刷在烘焙纸上
  2. 烤箱 150℃，10分钟
  3. 趁热弯曲成形状
  4. 冷却变脆

或：
  1. 细砂糖加少量水成糊状
  2. 烘焙纸上描字/图案
  3. 晾干过夜
```

### 5.3 焦糖网 (Caramelized Sugar)
```
工具：糖浆加热至软焦糖状态
方法：用勺子来回划出网格状
应用：Panna Cotta 风格鸡尾酒

⚠️ 焦糖温度极高，小心烫伤
```

## 六、可食花

```
季节性装饰，提升精致度：
| 花名 | 风味 | 用法 |
|------|------|------|
| 三色堇 | 无味 | 仅视觉装饰 |
| 金盏花 | 微苦 | 东南亚风格 |
| 玫瑰花瓣 | 微香 | 浪漫风格 |
| 薰衣草 | 花香 | 1朵即可，极香 |
| 可食樱草 | 微酸 | Spring cocktails |

来源：有机花店（确保无农药）或专业食品级可食花
警告：绝对不要用普通花店的花（可能有毒）
```

## 七、装饰决策速查表

```
鸡尾酒 → 装饰匹配原则：

马天尼类：
  - 橄榄 → 咸，经典
  - 洋葱 → Gibson
  - 柠檬皮 → 现代干式（推荐）

古典类：
  - 橙皮（火焰）→ 波本
  - 柠檬皮 → 黑麦
  - 橙皮 → 裸麦

酸酒类：
  - 无装饰（泡沫即装饰）→ 专业干净
  - 或：橙皮轮（高球杯）
  - 或：应季水果（桃子/草莓）

热带类：
  - 大量薄荷叶 → Mojito
  - 菠萝角+樱桃 → Mai Tai
  - 青柠角 → 大多数

高球类：
  - 柠檬/橙片 → Gin & Tonic
  - 黄瓜条 → Hendrick's
  - 迷迭香 → 苏格兰风格
```
