# 调味料与辅料 (Mixers & Extras)

> 基酒之外的一切液体 — 从气泡水到果汁，从糖浆到奶制品。
> 这一模块覆盖所有"配角"，让主配角都能发挥最佳状态。

---

## 一、气泡水类

### 1.1 碳酸饮料速查
| 名称 | 风味 | 基酒搭配 | 注意事项 |
|------|------|---------|---------|
| 苏打水 (Soda Water) | 无味（碳酸化） | 任何 | 最安全的配角 |
| 汤力水 (Tonic Water) | 奎宁+甜 | 金酒、伏特加 | 必须冰镇，冰的才起泡 |
| 姜汁啤酒 (Ginger Beer) | 生姜+气泡 | 伏特加、朗姆 | 辣度差异大，选喜欢的 |
| 可乐 (Cola) | 焦糖+可乐果 | 朗姆、威士忌 | 甜，掩盖劣质酒 |
| 干姜水 (Dry Ginger Ale) | 轻微姜味 | 波本、威士忌 | 比姜汁啤酒温和 |
| 苏打水(Italian) | 无味极细泡 | 意式 Spritz | San Pellegrino 首选 |
| 莱姆苏打 (Lime Soda) | 青柠味 | 龙舌兰 | 常用于Paloma |
| 通宁水(Tonic) | 苦+甜+气泡 | 金酒 | 有无糖版本 |

### 1.2 汤力水深度指南
```
为什么汤力水对 G&T 至关重要？
  - 汤力水是苦的（奎宁），中和金酒的杜松子
  - 甜平衡酸度
  - 气泡提供清爽感

选择建议：
  - Fever-Tree：高级，奎宁纯正，入门首选
  - 屈臣氏：性价比高，国内易买
  - 怡泉(Schweppes)：大众选择
  - 不要用：无糖节食版（甜度不对）

⚠️ 永远用冰的汤力水，温热的没有灵魂
```

## 二、果汁类

### 2.1 柑橘类（必须鲜榨）
| 果汁 | 特点 | 保存 | 配对 |
|------|------|------|------|
| 青柠汁 | 最常用、酸强、微苦 | 鲜榨，24h内用完 | 龙舌兰、朗姆 |
| 柠檬汁 | 温和酸、微香 | 鲜榨，24h内用完 | 威士忌、金酒 |
| 橙汁 | 甜酸适中 | 鲜榨，12h内用完 | 伏特加、朗姆 |
| 西柚汁 | 苦甜并存 | 鲜榨，24h内用完 | 龙舌兰（金巴利）|
| 葡萄柚汁 | 更苦、更独特 | 鲜榨，24h内用完 | 龙舌兰（金巴利）|

```
⚠️ 绝对规则：不用瓶装/罐装柑橘汁做鸡尾酒
  - 瓶装柠檬汁 = 三秒柠檬酸（太酸、太苦）
  - 鲜榨 = 自然的酸甜平衡
  - 除非特殊情况，否则永远鲜榨
```

### 2.2 果汁保存技巧
```
鲜榨果汁保存期：
  - 柑橘类：24小时（最佳），最多48小时（风味下降）
  - 热带果汁（菠萝/芒果）：48小时
  - 浆果类（蔓越莓/石榴）：24小时（易氧化变色）
  - 苹果/梨：48小时（尽快用）

延缓氧化：
  - 加少量柠檬汁（酸的抗氧化）
  - 盖紧，冷藏
  - 不透明的瓶子比透明的好
```

### 2.3 其他果汁
| 果汁 | 风味 | 配对 |
|------|------|------|
| 菠萝汁 | 热带、甜 | 朗姆、伏特加 |
| 芒果汁 | 浓郁、甜 | 伏特加、朗姆 |
| 蔓越莓汁 | 酸、颜色深 | 伏特加、金酒 |
| 石榴汁 | 甜、酸、深红 | 伏特加 |
| 番茄汁 | 咸、鲜 | 伏特加（血腥玛丽）|
| 苹果汁 | 清淡、甜 | 波本、裸麦 |
| 梨汁 | 精致、花香 | 金酒、伏特加 |

## 三、糖浆类

### 3.1 基础糖浆
```
简单糖浆 (Simple Syrup) — 1:1
  做法：白砂糖:水 = 1:1，煮至溶解，冷却
  保存：室温2周/冷藏1个月

浓糖浆 (Rich Simple Syrup) — 2:1
  做法：白砂糖:水 = 2:1
  优点：不易结晶，甜度更高
  用途：酸味酒（糖分需求更大）

有机蜂蜜糖浆 (Honey Syrup) — 2:1
  做法：蜂蜜:水 = 2:1，微波/加热搅拌
  注意：冷水无法溶解蜂蜜
  替代：直接用蜂蜜摇和（但溶解不均）

龙舌兰花蜜 (Agave Nectar)
  特点：比糖浆更接近液体的天然甜
  用途：龙舌兰酒调酒（风味匹配）
  注意：极甜，1/2 用量即可
```

### 3.2 风味糖浆
| 糖浆 | 做法 | 配对 |
|------|------|------|
| 香草糖浆 | 香草荚剖开 + 1:1 糖浆，浸泡24h | 任何需要甜味的酒 |
| 生姜糖浆 | 新鲜姜片 + 1:1 糖浆，浸泡3天 | 威士忌酸、老古典 |
| 辣椒糖浆 | 墨西哥小辣椒 + 1:1 糖浆，浸泡1周 | 龙舌兰、伏特加 |
| 薄荷糖浆 | 薄荷叶 + 1:1 糖浆，浸泡24h（不要捣烂） | 朗姆、Mojito |
| 玫瑰糖浆 | 玫瑰花瓣 + 1:1 糖浆，浸泡24h | 金酒、花香鸡尾酒 |
| 南瓜派糖浆 | 南瓜泥+香料+1:1 糖浆，煮20分钟 | 波本古典 |
| 榛果糖浆 | 烤榛子 + 2:1 糖浆，浸泡1周（滤渣） | 咖啡鸡尾酒 |

### 3.3 商业糖浆推荐
| 品牌 | 特点 | 适合场景 |
|------|------|---------|
| Monin | 品种最全，品质稳定 | 专业吧台 |
| Giffard | 法国，果味纯正 | 高端酒吧 |
| Liber | 有机，糖分更低 | 健康定位酒吧 |
| 自制 | 成本低，新鲜 | 家用/小批量 |

## 四、奶制品与蛋类

### 4.1 全脂牛奶 (Whole Milk)
```
用途：
  - 澄清鸡尾酒（Milk Punch）
  - 蛋酒（Eggnog）
  - 热酒精饮品的基底

关键：全脂 > 脱脂（全脂蛋白质更丰富，澄清效果更好）
```

### 4.2 淡奶油 (Light Cream)
```
用途：
  - 新加坡司令（辅助泡沫）
  - 蛋酒
  - 丝绒感收尾

用法：先摇和，再轻轻倒入（保持分层效果）
```

### 4.3 蛋清 (Egg White)
```
用途：制作细腻泡沫（酸酒类）
关键要点：
  - 先干摇（无冰）15秒乳化
  - 再加冰摇15秒降温
  - 双重过滤（避免蛋液碎屑入杯）
  - 顶层淋苦精（标准出品方式）

保存：
  - 整蛋：冷藏可存 3 周（检查日期）
  - 蛋清：分装冷冻可存 3 个月（解冻后尽快用）
  - 巴氏消毒蛋液：更安全，超市有售

⚠️ 安全提示：沙门氏菌风险极低但存在，高风险人群（孕妇/免疫低下）可用巴氏消毒蛋液替代
```

### 4.4 椰子奶油 (Coconut Cream)
```
用途：Pina Colada、热带鸡尾酒
注意：不是椰子水（无脂），是椰子肉打碎加水的浓稠液体
代表品牌：Coco Lopez（业界标准）
替代：椰浆（稍稀）加椰子油
```

## 果汁/糖浆新鲜度速查表

| 原料 | 新鲜状态 | 可接受状态 | 必须丢弃 |
|------|---------|-----------|---------|
| 青柠/柠檬汁 | 清亮、酸香 | 微浑 | 颜色变深/发苦 |
| 橙汁 | 橙黄、清香 | 轻微变色 | 发酵味 |
| 菠萝汁 | 金黄、微甜 | 轻微变色 | 发酵/起泡 |
| 简单糖浆 | 清澈透明 | — | 白色浑浊/起霉 |
| 香草糖浆 | 棕黄、香草味 | — | 无香草味/变质味 |
| 苏打水/汤力水 | 气泡充足 | 气泡减弱 | 无气泡 |
| 牛奶/蛋清 | 正常 | — | 任何异味/变色 |

> ⚠️ 适量饮酒，禁止酒驾。
