# 利口酒与苦精 (Liqueurs & Bitters)

> 利口酒 = 烈酒 + 糖 + 风味物质。苦精 = 高浓度植物药草提取物。
> 这两类是鸡尾酒的"调味品"，掌握它们才能真正进入调酒的自由世界。

---

## 一、利口酒 (Liqueurs)

### 1.1 利口酒 vs 白兰地 vs 烈酒
```
核心区别：
  烈酒：蒸馏产物，无添加糖
  利口酒：蒸馏酒 + 糖 + 风味（调味）

糖分标准：
  - 法国：利口酒必须含糖 ≥ 100g/L
  - 美国：通常 ≥ 2.5% 糖分
  - 一些干型利口酒（如干邑）不在此列

两大类：
  1. 天然利口酒：水果/草本/香料直接浸泡
  2. 合成利口酒：使用人工香料/色素（不推荐）
```

## 二、味美思 (Vermouth)

### 2.1 味美思是什么
```
定义：强化葡萄酒（葡萄酒+烈酒）+ 草本调味
历史：意大利/法国医生（Vermouth = wormwood，德语"Wermut"苦艾）
关键：它是加味葡萄酒，不是蒸馏酒，所以必须冷藏
```

### 2.2 四大类型

| 类型 | 风味 | 经典用途 | 推荐品牌 |
|------|------|---------|---------|
| Dry Vermouth | 极干、草本、清冽 | Dry Martini | Noilly Prat（法国）、Dolin（法国） |
| Sweet (Rosso) | 焦糖、香料、甜 | Manhattan | Martini Rosso、Carpano Antica |
| White/Bianco | 花香、微甜 | 非传统马天尼变体 | Carpano Bianco、Dolin Blanc |
| Extra Dry | 几乎无甜 | 极干 Martini | Noilly Prat Extra Dry |

### 2.3 味美思考核清单
```
使用前必须检查：
  □ 开瓶多久了？（超过4周？倒掉）
  □ 保存方式？（必须冷藏，直立）
  □ 闻起来对吗？（氧化味=坏了）
  □ 颜色对吗？（变深=氧化）

购买建议：
  - 买小瓶（375ml）而非大瓶，喝完再买
  - 多试几个品牌，找到自己的偏好
  - Noilly Prat + Martini = 经典组合
```

## 三、苦精 (Bitters)

### 3.1 苦精是什么
```
定义：高浓度植物药草/香料提取物，酒精溶剂，极少量使用
历史：原始药用，后成为鸡尾酒调味剂
使用量：通常以 "dash" 计算（≈0.5ml/ dash）
核心原则：苦精是调味，不是主料（除了特殊酒款）
```

### 3.2 苦精速查表

| 名称 | 风味 | 核心用途 | 代表品牌 |
|------|------|---------|---------|
| Angostura (安高天娜) | 丁香、香料、微微辣 | 古典、曼哈顿、所有经典 | Angostura |
| Peychaud's | 茴香、樱桃、微甜 | Sazerac | Peychaud's |
| Orange Bitters | 柑橘皮 | Martini 升级、各种 | Fee Brothers / Angostura |
| Celery Bitters | 芹菜 | 血腥玛丽 | Fee Brothers |
| Chocolate Bitters | 可可、咖啡 | Old Fashioned 变体 | Fee Brothers / Scrappy's |
| Aromatic Bitters | 综合草本 | 万能调味 | Angostura |
| Chocolate Bitters (浓郁) | 深度可可 | 巧克力鸡尾酒 | Bittermens / Raine |

### 3.3 苦精使用技巧
```
1 dash = 甩瓶子的一下（约 0.5ml）
2 dashes = 甩两下
1 bar spoon = 约 5ml

苦精的隐藏用法：
  - 提鲜：任何不够"活"的鸡尾酒，加1 dash
  - 平衡：甜+酸+苦是永恒三角
  - 结束：顶层淋苦精是 Pisco Sour、Old Fashioned 的标准收尾
  - 增香：0.5ml 糖浆 + 2 dashes = "苦精糖浆"

苦精品牌建议：
  - 必买：Angostura + Peychaud's + Orange Bitters
  - 进阶级：Chocolate + Celery
  - 不要贪多：先精通三种，再扩展
```

## 四、自制利口酒与苦精

### 4.1 自制糖浆基础 (Infused Simple Syrup)
```
自制接骨木花利口酒：
  1. 接骨木花（干）20g + 伏特加 200ml
  2. 浸泡 48 小时，过滤
  3. 加 1:1 糖浆 200ml
  4. 冷藏可存 3 个月

自制香草利口酒：
  1. 香草荚 3根剖开 + 伏特加 300ml
  2. 浸泡 1 周（每天摇一次）
  3. 加 1:1 糖浆 150ml
  4. 过滤，瓶装

自制苦精（基础版）：
  1. 安高天娜苦精 50ml（作底）
  2. 金酒 100ml + 丁香 5粒 + 八角 2个 + 肉桂棒 1根
  3. 浸泡 48 小时
  4. 滴入几滴于任何鸡尾酒中
```

### 4.2 家制苦精配方
```
基础草本苦精：
  香料：
    - 丁香 3粒
    - 八角 1个
    - 肉桂棒 1小段
    - 白胡椒 5粒
  草本：
    - 干苦橙皮 5g
    - 干柠檬皮 3g
    - 龙胆根 2g（可选，巨苦）
  液体：
    - 中性伏特加 200ml（40%）
  
  做法：
    1. 所有香料碾碎（不要磨成粉）
    2. 入罐，加伏特加
    3. 每天摇一次，浸泡 2 周
    4. 用咖啡滤纸过滤（很慢，耐心）
    5. 加 1:1 糖浆 20ml（调味）
    6. 瓶装，冷藏可存 6 个月

用法：从 1 dash 开始测试
```
