# 基酒完全指南 (Spirits)

> 六大基酒 + 国产白酒，一文读懂每一瓶酒。

---

## 一、伏特加 (VODKA)

### 1.1 核心特性
```
定义：以谷物/薯类为原料，经多次蒸馏至极高纯度，无色无味（理论上）
关键概念："中性"是相对的，不同伏特加的顺滑度和余韵差异很大
酒精：40% ABV（标准）
```

### 1.2 分类与代表
| 类型 | 代表品牌 | 风格 | 价格 |
|------|---------|------|------|
| 俄罗斯/东欧传统 | 绝对、斯托利、灰雁 | 中性、干净 | ¥100-200 |
| 瑞典/芬兰 | 绝对、芬恩 | 圆润、微甜 | ¥120-200 |
| 美国 | 天空、皇冠 | 轻盈 | ¥80-150 |
| 精酿/超高端 | 诗洛珂、倒怪 | 独特处理 | ¥300+ |
| 风味伏特加 | 柠檬伏特加、辣椒伏特加 | 加味 | 不推荐 |

### 1.3 调酒应用
```
最佳搭档：
  - 马天尼（伏特加版）：口感更顺滑
  - 莫斯科骡子：姜+酸激活风味
  - 大都会：蔓越莓+君度
  - 螺丝刀：橙汁掩盖酒精感

不适用：
  - 马天尼（金酒版更正宗）
  - 需要风味骨架的复杂鸡尾酒
```

## 二、金酒 (GIN)

### 2.1 核心特性
```
定义：谷物烈酒 + 杜松子为主的一系列植物调料（"botanicals"）
关键成分：杜松子（必须）+ 芫荽籽 + 柑橘皮 + 鸢尾根 + 其他
酒精：40% ABV（标准）
核心哲学："调酒师的画布"——金酒的风味决定了鸡尾酒的骨架
```

### 2.2 分类详解

| 类型 | 特点 | 代表品牌 | 调酒推荐 |
|------|------|---------|---------|
| London Dry | 干、清晰、最常见 | 哥顿、必富达、添加利 | Negroni、Martini、G&T |
| Plymouth | 更饱满、微甜、杜松子更强 | Plymouth | Devon Sours |
| Old Tom | 轻微甜，比Dry更历史感 | Hayman's | Tom Collins |
| Navy Strength | 57%ABV以上，风味更浓 | Plymouth Navy | 任何需要强劲风味的酒 |
| 新西式/当代 | 打破规则，水果/草本多样 | 六金酒、植物学家、Hendrick's | 现代创意鸡尾酒 |
| Geneva/Jenever | 荷兰风格，麦芽味浓 |琴酒 | 古典鸡尾酒 |

### 2.3 调酒应用
```
核心酒款：
  - Negroni：London Dry 最佳
  - Gin & Tonic：添加新鲜 botanicals
  - Aviation：需要 Maraschino + Violet
  - Singapore Sling：London Dry
  - Bee's Knees：Plymouth 或当代风格

金酒选择决策：
  - 不知道选哪个 → 哥顿（不会错）
  - 想要现代感 → 六/植物学家
  - 想要顺滑 → Plymouth
  - 想要花香 → Hendrick's
```

## 三、朗姆酒 (RUM)

### 3.1 核心特性
```
定义：以甘蔗为原料（甘蔗汁或糖蜜），发酵蒸馏
关键复杂度：不同产区/陈年差异极大，从清淡到浓郁
酒精：40% ABV（标准）
分类：白/金/黑/陈年/农业朗姆
```

### 3.2 分类与代表
| 类型 | 特点 | 代表 | 调酒推荐 |
|------|------|------|---------|
| 白朗姆 | 轻盈、无色、清新 | 百加得白、哈瓦那俱乐部3年 | Daiquiri、Mojito |
| 金朗姆 | 轻微陈年、琥珀色 | 船长摩根、百加得金色 | Mai Tai（部分）、Long Island |
| 黑朗姆 | 深色、焦糖、浓郁 | 迈尔船长(Myer's)、Goslings | Dark 'n' Stormy、Bahama Mama |
| 陈年朗姆 | 真正陈年，复杂 | 外交官、巴巴多斯10年 | 古典鸡尾酒 |
| 农业朗姆 | 甘蔗汁，独特草本味 | Clairin（海地）、Rhum J.M | 进阶调酒 |

### 3.3 调酒应用
```
热带系：白朗姆 + 果汁 → Mojito、Daiquiri、Pina Colada
古典系：陈年朗姆 → Old Fashioned 变体
加勒比风格：黑朗姆 + 姜汁 → Dark 'n' Stormy
```

## 四、龙舌兰 (TEQUILA)

### 4.1 核心特性
```
定义：100% 蓝色韦伯龙舌兰 (Blue Weber Agave) 蒸馏
关键规则：MIXTO（含51-99%龙舌兰）≠ 100% Agave
酒精：40% ABV（标准）
核心原则：永远选 100% Agave
```

### 4.2 分类详解
| 类型 | 陈化时间 | 风味 | 推荐品牌 |
|------|---------|------|---------|
| Blanco / Silver | 0-2个月 | 最纯净、龙舌兰香最浓 | 豪帅白、奥乔银 |
| Reposado | 2-12个月 | 轻微木质、圆润 | 唐胡里奥Reposado、埃尔托罗 |
| Añejo | 1-3年 | 深度木质、复杂 | Herradura Añejo |
| Extra Añejo | 3年+ | 极复杂、接近威士忌 | 雷博士(Clase Azul) |

### 4.3 调酒应用
```
Blanco（最常用）：
  - Margarita（必须用 Blanco）
  - Paloma
  - Tequila Sunrise
  - 自制Mezcal替代可出变体

Reposado：
  - Oaxacan Old Fashioned
  - Reposado Manhattan 变体
  - 烟熏版本

⚠️ 不要用龙舌兰做：Negroni（意大利经典，不换）
✅ 大胆尝试：Tequila Negroni（现代变体，圈内认可）
```

## 五、威士忌 (WHISKEY / WHISKY)

### 5.1 全球分类速查
| 类型 | 产地 | 风味 | 调酒推荐 |
|------|------|------|---------|
| 波本 (Bourbon) | 美国 | 甜润、香草、玉米 | Old Fashioned、Manhattan |
| 黑麦 (Rye) | 美国/加拿大 | 辛辣、香料、黑麦香 | Manhattan（经典）、Sazerac |
| 田纳西 | 美国 | 柔和、木炭过滤 | 与波本类似 |
| 苏格兰 | 苏格兰 | 烟熏、泥煤、果香 | Penicillin、Rob Roy |
| 爱尔兰 | 爱尔兰 | 顺滑、三次蒸馏 | Irish Coffee、Irish Sour |
| 日本 | 日本 | 细致、果香、轻泥煤 | Japanese Highball |
| 加拿大 | 加拿大 | 轻盈、多样 | 多用于调酒 |

### 5.2 调酒应用速查
```
鸡尾酒用什么威士忌？

Old Fashioned → 波本（甜润）
  入门：金宾 (Jim Beam)
  进阶级：四玫瑰(4 Roses)、布齐(Bulleit)

Manhattan → 黑麦（传统）或波本（美式）
  黑麦 = 更干、更辛辣
  波本 = 更甜、更圆润

Sazerac → 裸麦(Rye)（必须）或黑麦
  裸麦更传统，波本更现代

Penicillin → 调和苏格兰（核心）+ Islay 艾莱岛（顶层）
  关键：艾莱岛威士忌的烟熏味

Japanese Highball → 日本威士忌
  山崎、响、秩父、三得利
```

## 六、白兰地 (BRANDY)

### 6.1 分类
| 类型 | 特点 | 代表品牌 |
|------|------|---------|
| 干邑 (Cognac) | 法国产区，严格法规 | 轩尼诗、人头马、马爹利 |
| 雅文邑 (Armagnac) | 法国另一产区，更粗犷 | 达尔丘(Delord) |
| 皮斯科 (Pisco) | 秘鲁/智利，葡萄蒸馏 | Pisco Porton、Control C |
| 水果白兰地 | 苹果、梨、梅子 | 卡尔瓦多斯(Calvados) |

### 6.2 调酒应用
```
Sidecar → 干邑
  入门：轩尼诗VS
  升级：VSOP、XO

Pisco Sour → 皮斯科
  必须用 Pisco，别的白兰地做不出那个味道
  推荐：Pisco Porton

Brandy Crusta → 干邑古典
  早期马天尼的雏形
```

## 七、中国白酒 (BAIJIU)

### 7.1 香型分类
| 香型 | 风味 | 代表品牌 | 调酒建议 |
|------|------|---------|---------|
| 酱香型 | 酱油、醇厚、复杂 | 茅台、郎酒、习酒 | 适合纯饮，调酒偏难 |
| 浓香型 | 窖香、绵甜 | 五粮液、泸州老窖、剑南春 | 可做白酒 Sour |
| 清香型 | 清雅、乙酸乙酯 | 玻汾、红星二锅头 | 最适合调酒（最干净） |
| 凤香型 | 清浓兼有 | 西凤酒 | 独特，可尝试 |
| 米香型 | 米香、蜜香 | 桂林三花酒 | 轻盈，适合热带调酒 |

### 7.2 白酒调酒指南
```
为什么白酒难调酒？
  1. 风味化合物太复杂（数百种酯类）
  2. 大多数鸡尾酒的风味会被白酒盖住
  3. 酒精感强烈，需要强酸强甜平衡

成功策略：
  ✓ 清香型基酒（玻汾/红星二锅头）作为练习首选
  ✓ 强酸（柠檬/青柠）+ 强甜（糖浆）
  ✓ 中草药苦精辅助（味美思/Aperol）
  ✓ 稀释（摇和 + 冰块）降低刺激感

推荐配方：见 recipes/signature.md「白酒酸酒」
```

## 基酒保存与选购

```
开瓶后保存：
  - 烈酒（40%+）：阴凉处直立保存，1-2年风味稳定
  - 味美思/开胃酒：必须冷藏，2-4周用完
  - 糖浆/利口酒：冷藏，3-6个月

购酒预算（调酒师推荐）：
  - 入门：¥80-150/瓶（练习用，不心疼）
  - 进阶级：¥150-300/瓶（日常出品）
  - 专业级：¥300+/瓶（高端酒款）

切记：
  ❌ 不要用你不愿意直接喝的酒做鸡尾酒
  ✅ 基酒决定上限，用好酒做出好酒的概率更高
```

> ⚠️ 适量饮酒，禁止酒驾。
