## 模块 1：六大基酒深度认知

### 1.1 伏特加 (Vodka)
| 维度 | 知识点 |
|------|--------|
| 原料 | 谷物（小麦、黑麦）或薯类 |
| 风格 | 中性 vs 风味型 |
| 产地 | 俄罗斯/波兰（传统）vs 瑞典/法国（新世界） |
| 入手推荐 | 绝对（中性）、灰雁（高端中性）、野牛草（风味） |
| 经典搭配 | 伏特加马天尼、莫斯科骡子、大都会 |

### 1.2 金酒 (Gin)
| 维度 | 知识点 |
|------|--------|
| 核心 | 杜松子为主要风味 |
| 分类 | London Dry / Old Tom / Plymouth / Navy Strength / 新西式 |
| 入手推荐 | 哥顿（基础）、必富达（经典Dry）、亨利（高端）、六（新式） |
| 关键概念 | "金酒是调酒师的画布" — 它的风味骨架决定整杯酒的走向 |

### 1.3 朗姆酒 (Rum)
| 维度 | 知识点 |
|------|--------|
| 原料 | 甘蔗汁 / 糖蜜 |
| 分类 | 白 / 金 / 黑 / 陈年 / 农业朗姆(Rhum Agricole) |
| 入手推荐 | 百加得白（入门白）、船长金（入门金）、外交官7年（入门陈年） |
| 经典搭配 | 莫吉托、代基里、迈泰 |

### 1.4 龙舌兰 (Tequila)
| 维度 | 知识点 |
|------|--------|
| 原料 | 蓝色韦伯龙舌兰 |
| 分类 | Blanco / Reposado / Añejo / Extra Añejo |
| 核心区别 | 100% Agave vs Mixto（务必选 100%） |
| 入手推荐 | 豪帅白（入门Blanco）、唐胡里奥Reposado（入门陈年） |

### 1.5 威士忌 (Whisky/Whiskey)
| 维度 | 知识点 |
|------|--------|
| 大类 | 苏格兰 / 爱尔兰 / 美国 / 日本 / 加拿大 |
| 调酒首选 | 波本（甜润，调酒友好）、黑麦（辛辣，经典鸡尾酒标配） |
| 入手推荐 | 金宾（波本基础）、杰克丹尼（田纳西）、猿（日本入门） |
| 经典搭配 | 古典、曼哈顿、威士忌酸 |

### 1.6 白兰地 (Brandy)
| 维度 | 知识点 |
|------|--------|
| 分类 | 干邑(Cognac) / 雅文邑(Armagnac) / 水果白兰地 |
| 调酒用法 | 侧车(Sidecar)、白兰地亚历山大 |
| 入手推荐 | 轩尼诗VS（入门干邑） |

## 模块 2：搅拌法 (Stirring)

### 2.1 什么时候搅拌 vs 摇和？

| 条件 | 方法 | 原因 |
|------|------|------|
| 全透明原料（酒+酒） | 搅拌 | 保持清澈透明 |
| 含果汁/奶/蛋 | 摇和 | 必须充分乳化混合 |
| 需要丝绒口感 | 搅拌 | 温和稀释，质感细腻 |
| 需要丰富泡沫 | 摇和 | 空气混入产生泡沫 |

### 2.2 搅拌手法
```
1. 调酒杯加冰（大冰块优先，减少碎冰融化）
2. 倒入原料
3. 吧勺插入杯中，指腹夹住勺柄
4. 手腕发力，沿杯壁画圆搅拌
5. 20-30秒，手感杯壁冰凉即到位
6. 滤入冰镇酒杯
```

### 2.3 第一杯搅拌酒 — Dry Martini（干马天尼）
```
配方：
  金酒          60ml
  干味美思      10ml
  橄榄或柠檬皮  装饰

步骤：
  1. 调酒杯加冰
  2. 倒入金酒和味美思
  3. 搅拌 25 秒
  4. 滤入冰镇马天尼杯
  5. 拧柠檬皮（喷洒精油）或投入橄榄

争议点：
  - 搅拌 vs 摇和？IBA 标准是搅拌。邦德说"shaken, not stirred"是错的（对马天尼而言）
  - 味美思比例？从 1:1（湿）到只喷一下（超干），看你喜好
```

## 模块 3：经典鸡尾酒 10 杯

每杯包含：配方、步骤、关键技巧、常见错误

### 3.1 Margarita（玛格丽特）
```
配方：
  龙舌兰 Blanco  50ml
  君度          20ml
  青柠汁        20ml
  盐边          可选

关键：青柠必须鲜榨，三秒柠檬（瓶装）做出来完全不同
```

### 3.2 Mojito（莫吉托）
```
配方：
  白朗姆          50ml
  青柠汁          30ml
  糖浆            20ml
  薄荷叶          8-10片
  苏打水          适量

关键：薄荷叶轻拍（拍醒），不要捣碎（会出苦味）
```

### 3.3 Old Fashioned（古典）
```
配方：
  波本威士忌      60ml
  安高天娜苦精    2 dashes
  糖             1块 或 糖浆10ml
  橙皮           装饰

关键：搅拌要耐心，30秒起步，让冰充分稀释
```

### 3.4 Daiquiri（代基里）
```
配方：
  白朗姆          60ml
  青柠汁          25ml
  糖浆            20ml

关键：摇和后要滤入冰镇 coupe 杯，不加冰。这是检验调酒师基本功的酒。
```

### 3.5 Manhattan（曼哈顿）
```
配方：
  黑麦威士忌      50ml
  甜味美思        20ml
  安高天娜苦精    2 dashes

关键：搅拌法，不用摇。甜/干味美思比例决定风格。
```

### 3.6 Negroni（尼格罗尼）
```
配方：
  金酒            30ml
  金巴利          30ml
  甜味美思        30ml

关键：1:1:1 是最优雅的比例。搅拌法，橙皮装饰。
```

### 3.7 Whiskey Sour（威士忌酸）
```
配方：
  波本威士忌      50ml
  柠檬汁          25ml
  糖浆            20ml
  蛋清（可选）    1个

关键：加蛋清就摇两次——先干摇（无冰）15秒，再加冰摇15秒
```

### 3.8 Cosmopolitan（大都会）
```
配方：
  柑橘伏特加      40ml
  君度            15ml
  青柠汁          15ml
  蔓越莓汁        30ml

关键：摇和，滤入冰镇 coupe 杯，火焰橙皮装饰
```

### 3.9 Moscow Mule（莫斯科骡子）
```
配方：
  伏特加          50ml
  青柠汁          15ml
  姜汁啤酒        120ml

关键：铜杯是传统容器（姜汁啤酒的微反应提升风味感知）
```

### 3.10 Espresso Martini（浓缩咖啡马天尼）
```
配方：
  伏特加          40ml
  咖啡利口酒      20ml
  浓缩咖啡        30ml
  糖浆            10ml

关键：咖啡必须新鲜萃取，浓缩的。速溶的泡沫不对。
```
