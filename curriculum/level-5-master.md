## 模块 1：大师级技法

### 1.1 精确控温 (Temperature Control)
```
冰镇不只是"加冰"：
  - 冰镇杯子：提前 30 分钟放入冷冻室
  - 冰镇调酒壶：壶身冷冻后再用，减少融化稀释
  - 双重冷却：酒液先冰镇再调味
  - 温度与感知的数学：
      0°C 冰镇 → 香气封闭
      5-8°C 理想 → 香气开放+口感圆润
      12°C+    → 酒精刺鼻，感知偏烈
```

### 1.2 香气管理 (Aromatics)
```
鸡尾酒三感：视觉 / 味觉 / 嗅觉
香气往往被忽视，却是记忆最深的感官：

香气来源层次：
  第一层：装饰（橙皮、柠檬皮）— 入口前的第一感知
  第二层：基酒本身（杜松子、烟熏）
  第三层：辅助香气（香料、香草）

提升香气的技术：
  - 火焰橙皮：热油雾化，精油扩散
  - 香草轻烤：迷迭香/百里香用火苗过一下
  - 杯口摩挲：肉桂粉在杯口摩擦出香
  - 冷萃 vs 热萃：冷萃保留花香，热萃保留焦香
```

### 1.3 减法艺术 (Subtractive Mixology)
```
大师原则：最好的一杯酒，是去掉所有不必要的东西

减法清单：
  □ 这款装饰有必要吗？— 多余的装饰等于噪音
  □ 这三种甜味来源是否重复？— 糖浆+利口酒+果汁都有甜
  □ 冰块的形状是否匹配这杯酒？— 方冰 vs 球冰 vs 碎冰
  □ 5 种原料能否变成 4 种？
  □ 4 种能否变成 3 种？

练习：拿一款你做过的 6 原料鸡尾酒，删除 2 种，重新平衡。
```

### 1.4 时间维度的鸡尾酒 (Time-Based Cocktails)

| 类型 | 陈化时间 | 代表 | 技术 |
|------|---------|------|------|
| 快速浸泡 | 30 分钟 - 2 小时 | 伏特加浸香草 | 常温/低温浸泡 |
| 鸡尾酒陈化 | 1-4 周 | barrel-aged Negroni | 橡木桶/小木片陈化 |
| 瓶陈化 | 2-6 个月 | barrel-aged Manhattan | 小橡木桶 |
| 即饮型 | 出品即饮 | 绝大多数 | 速度与精准 |

**桶陈鸡尾酒操作规范：**
```
1. 选桶：5-25 升新橡木桶，或用波本/雪莉桶
2. 配比：酒液占桶容 80-90%（留空间让空气接触）
3. 周期：每 3 天取样品评
4. 结束：目标达到后，瓶装冷藏保存
5. 记录：每个阶段的风味变化必须记录

典型陈化时间：
  - Manhattan 类：1-2 周
  - Negroni 类：3-4 周
  - Old Fashioned 类：1-2 周
```

## 模块 2：品牌与哲学

### 2.1 调酒哲学三大流派

| 流派 | 核心信念 | 代表调酒师 |
|------|---------|-----------|
| 经典守护者 | 经典就是最好的，任何改动都是亵渎 | — |
| 现代主义者 | 在经典框架上，用技术/风味/视觉重新演绎 | Alex Kratena, Alex Day |
| 先锋实验派 | 打破一切规则，用科学和艺术重构鸡尾酒 | Dave Arnold, Grant Achatz |

### 2.2 你的调酒哲学宣言
```
问自己这三个问题，写下答案：

1. 你认为鸡尾酒的核心是什么？
   例："在可控的混乱中创造秩序" — Dale DeGroff

2. 你拒绝做什么？
   例：不做网红酒、不用劣质原料、不为销量牺牲品质

3. 你希望客人喝完你的酒后感受到什么？
   例："像认识了一个有趣的人" / "看到了一段风景"

这三个答案，就是你的调酒师品牌 DNA。
```

## 模块 3：行业文化与历史

### 3.1 鸡尾酒历史时间线
```
1780s  原始鸡尾酒（烈酒+糖+水）概念出现
1860s  黄金时代：Martini, Manhattan, Old Fashioned 诞生
1862  Jerry Thomas 出版《Barkeepers Manual》— 调酒师圣经
1880s  调酒师职业化开始（纽约、伦敦）
1900s  禁酒令时期（1920-1933）：地下酒吧 speakeasy 文化
1933  禁酒令废除，调酒业复苏
1940s-60s  战后繁荣，马天尼文化，朗姆酒热潮
1970s-80s  调酒业低谷，瓶装鸡尾酒主导
1990s  经典鸡尾酒复兴（伦敦、纽约）
2000s  精品金酒革命 + 分子调酒
2010s  内格罗尼革命，全球鸡尾酒文化爆发
2020s  低碳、无酒精/低酒精、可持续
```

### 3.2 Speakeasy 文化与传承
```
禁酒令时期的 speakeasy：
  - 地下酒吧需要暗号才能进入
  - 调酒师是艺术家也是骗子
  - "Jimmy the bartender"——最受欢迎的虚构调酒师

现代 speakeasy 的意义：
  - 保留神秘感和仪式感
  - 让调酒回归手工艺
  - 成为一种文化符号，而非复古 cosplay

大师的 speakeasy 态度：
  不做假的怀旧，而做真的传承。
  经典的精神是创造，不是复制。
```

### 3.3 亚洲鸡尾酒文化
```
日本：职人精神
  - 冰的哲学：一块冰的透明度代表调酒师的尊严
  - 精确计量：微量成分影响整杯平衡
  - 代表人物：上田和男（Hard Shake 创始人）

中国：新中式调酒
  - 白酒鸡尾酒：基酒本土化
  - 中式风味：桂花、茉莉、陈皮、茅台
  - 代表酒吧：E.P.I.C（上海）、Sober Company（上海）

东南亚：热带主义
  - 拉茶技法（Teh Tarik Method）
  - 丰富草本：斑斓叶、香兰叶、青柠叶
  - 代表人物：Atsuo Suzuki, Jeff "B_half" Sax
```

## 模块 4：大师级品鉴

### 4.1 品鉴框架 (DEGUSTATION FRAMEWORK)
```
五感品鉴法：

1. 视觉 (Visual)
   - 清澈度 / 颜色深度 / 泡沫稳定性 / 挂杯

2. 嗅觉 (Nose) — 先闻再喝
   - 第一鼻：直接闻（酒精主导）
   - 第二鼻：轻摇后闻（香气打开）
   - 关键词：水果 / 香料 / 草本 / 化工 / 氧化

3. 味觉 (Palate)
   - 入口第一感：甜 / 酸 / 苦 / 咸 / 鲜（顺序）
   - 中段：主体风味展开
   - 尾韵：回味长度和层次（好酒回味 10 秒+）

4. 质地 (Mouthfeel)
   - 轻盈 / 油润 / 泡沫感 / 单宁感 / 涩感

5. 平衡 (Balance) — 最终判断
   - 酒精感是否突出？
   - 某一种味道是否压倒其他？
   - 香气与口感是否一致？
```

### 4.2 坏酒诊断表
| 症状 | 可能原因 | 修复方法 |
|------|---------|---------|
| 过苦 | 柑橘皮多、浸泡过久 | 减皮、加糖 |
| 过酸 | 柠檬/青柠汁不鲜榨 | 每次用鲜榨 |
| 过甜 | 糖浆过多 | 减半糖，增加酸 |
| 过烈 | 冰不够/没稀释 | 加冰、摇久一点 |
| 无味 | 原料过期/不新鲜 | 全部换新鲜原料 |
| 颜色混浊 | 冰不干净/器具不洁 | 换新冰、清洁工具 |
| 泡沫太快消散 | 蛋清量不够/没干摇 | 重做 |
