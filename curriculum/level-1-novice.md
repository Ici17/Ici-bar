## 模块 1：调酒是什么

### 1.1 鸡尾酒的定义
- 鸡尾酒 = 基酒 + 辅料 + 装饰，至少两种原料混合
- IBA（国际调酒师协会）官方定义与分类
- 六大基酒速览：伏特加、金酒、朗姆酒、龙舌兰、威士忌、白兰地

### 1.2 鸡尾酒的三大基本结构
| 结构 | 说明 | 代表 |
|------|------|------|
| Sour 酸酒 | 基酒 + 酸 + 甜 | Whiskey Sour |
| Highball 海波 | 基酒 + 碳酸 | Gin & Tonic |
| Martini 马天尼型 | 基酒 + 加味酒 | Dry Martini |

### 1.3 你需要的第一套工具
| 工具 | 用途 | 入门款价格 |
|------|------|-----------|
| 量酒器 (Jigger) | 精确计量 | ¥15-30 |
| 摇酒壶 (Cobbler Shaker) | 混合冷却 | ¥30-80 |
| 吧勺 (Bar Spoon) | 搅拌 | ¥10-20 |
| 滤冰器 (Hawthorne Strainer) | 过滤冰块 | ¥15-30 |
| 捣棒 (Muddler) | 捣碎水果/香料 | ¥15-30 |

> 💡 入门五件套总价约 ¥100-200，够用到中级。

## 模块 2：你的第一杯酒

### 2.1 Gin & Tonic（金汤力）— 最简单的入门款
```
配方：
  金酒          45ml
  汤力水        120ml
  青柠角        1个

步骤：
  1. 高球杯加满冰
  2. 倒入金酒
  3. 沿杯壁缓缓注入汤力水（保持气泡）
  4. 轻轻提拉搅拌 1-2 次
  5. 青柠角挤汁后投入杯中

关键：
  - 汤力水一定要冰的，温的会失气泡
  - "缓缓注入"不是倒，是让液体沿杯壁滑下去
```

### 2.2 Rum & Coke（朗姆可乐）— 永远不会失败
```
配方：
  白朗姆        50ml
  可乐          120ml
  青柠角        1个

步骤：
  1. 高球杯加满冰
  2. 倒入朗姆酒
  3. 注入可乐
  4. 轻提吧勺混合
  5. 青柠装饰
```

### 2.3 Screwdriver（螺丝刀）— 只有两种原料
```
配方：
  伏特加        45ml
  橙汁          120ml

步骤：
  1. 高球杯加冰
  2. 伏特加 → 橙汁
  3. 搅拌
```

## 模块 3：摇和入门

### 3.1 为什么要摇？
- 充分混合不互溶的液体（如果汁 + 酒）
- 快速降温
- 稀释（冰化水，让口感更圆润）

### 3.2 摇酒壶使用方法
```
1. 配料倒入摇酒壶
2. 加冰（填满壶的 2/3）
3. 盖上壶盖，食指抵住壶盖
4. 用力摇晃 10-15 秒（听声音：从叮叮当当变成闷闷的）
5. 用滤冰器滤入杯中
6. 另加冰块（如果配方要求）
```

### 3.3 你的第一杯摇和酒 — Vodka Lemon
```
配方：
  伏特加        45ml
  柠檬汁        30ml
  糖浆          15ml

步骤：
  1. 摇酒壶中加入伏特加、柠檬汁、糖浆
  2. 加冰，摇 10-15 秒
  3. 滤入冰镇马天尼杯或高球杯
  4. 柠檬皮装饰
```
