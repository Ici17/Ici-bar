## 模块 1：配方解构 — 为什么这些原料在一起？

### 1.1 鸡尾酒的味觉架构
```
        酸（Acid）
          ↑
   甜 ←——→ 苦
          ↓
        醇（Alcohol）
```

每款经典鸡尾酒都是这个四边形的平衡：
| 酒款 | 甜 | 酸 | 苦 | 醇 |
|------|:--:|:--:|:--:|:--:|
| Daiquiri | ●●● | ●●●○ | ○ | ●●● |
| Negroni | ●● | ○ | ●●●● | ●●●● |
| Old Fashioned | ●● | ○ | ●●● | ●●●● |
| Margarita | ●● | ●●● | ○ | ●●● |

### 1.2 替换公式 — 一通百通
| 原版 | 替换 | 新酒 |
|------|------|------|
| Daiquiri: 白朗姆 → 龙舌兰 | 基酒替换 | Margarita 变体 |
| Margarita: 君度 → 覆盆子利口酒 | 甜味来源替换 | Raspberry Margarita |
| Negroni: 金酒 → 波本 | 基酒替换 | Boulevardier |
| Manhattan: 黑麦 → 波本 | 基酒替换 | Perfect Manhattan(加干味美思) |
| Whiskey Sour: 波本 → 阿玛雷托 | 基酒+甜味替换 | Amaretto Sour |

### 1.3 "即兴创作"方法论
```
Step 1: 选结构（Sour / Manhattan型 / Highball / Flip / 等）
Step 2: 选基酒
Step 3: 加甜味元素（利口酒/糖浆）
Step 4: 加酸味元素（柑橘汁）
Step 5: 加调味/苦精（微调平衡）
Step 6: 试做 → 微调 → 记录
```

## 模块 2：兑和法 (Building)

### 2.1 什么是兑和？
直接在饮用杯中依次倒入原料，不换容器，不摇不搅（或极轻搅拌）。

### 2.2 适用场景
- Highball 类（金汤力、朗姆可乐）
- 层次分明的视觉款（B-52、龙舌兰日出）
- 不需要充分混合的酒

### 2.3 分层技法 — Pousse-Café 原理
```
原理：密度不同 → 重的在下，轻的在上
密度排序（常见）：
  甘蔗利口酒 > 咖啡利口酒 > 奶油利口酒 > 君度 > 伏特加

分层技巧：
  1. 最重的先倒
  2. 吧勺反扣，沿勺背缓缓倒入下一层
  3. 速度极慢，让液体"浮"上去
  4. 每层之间停顿 3 秒
```

### 2.4 实操 — B-52
```
配方（分层）：
  咖啡利口酒 (Kahlúa)    20ml  ← 最重，先倒
  百利甜 (Baileys)        20ml  ← 中间
  君度 (Cointreau)        20ml  ← 最轻，最后

步骤：
  1. 子弹杯中先倒咖啡利口酒
  2. 吧勺反扣，沿勺背缓缓倒百利甜
  3. 同法倒入君度
  4. 三层分明

进阶：顶层点火（小心操作，饮酒前熄灭）
```

## 模块 3：捣碎法 (Muddling) 深入

### 3.1 原则
| 原料 | 手法 | 原因 |
|------|------|------|
| 薄荷 | 轻拍 | 提取精油，不破坏叶片（破叶出苦味） |
| 柑橘块 | 中力捣压 | 提取果汁和果皮精油 |
| 浆果 | 重力捣碎 | 释放果汁和颜色 |
| 姜/辣椒 | 中力捣压 | 提取辛辣，不过度（会过辣） |
| 糖块 | 重力捣碎 | 溶解糖块 |

### 3.2 进阶实操 — Caipirinha（卡匹林纳）
```
配方：
  卡莎萨（Cachaça）   60ml
  青柠               半个，切块
  糖                 2茶匙

步骤：
  1. 青柠块放入岩石杯
  2. 撒糖，用捣棒中力压出果汁
  3. 加碎冰至半杯
  4. 倒入卡莎萨
  5. 用吧勺上下提拉混合
  6. 补碎冰至满

关键：碎冰是灵魂，方冰做不出巴西的味道
```

## 模块 4：冰的学问

### 4.1 冰的四种形态
| 形态 | 用途 | 制备 |
|------|------|------|
| 大冰块 (2×2 inch) | 搅拌法、古典杯 | 冰格冻制 |
| 标准冰块 | 摇和法、Highball | 制冰机/冰格 |
| 碎冰 | 热带酒款、Julep | 冰袋敲碎或碎冰机 |
| 冰球 | 威士忌纯饮 | 冰球模具 |

### 4.2 稀释率 (Dilution)
- 摇和：约 25-30% 稀释
- 搅拌：约 15-20% 稀释
- 兑和：约 5-10% 稀释
- **稀释不是敌人，是配方的一部分** — 不稀释的鸡尾酒太烈太尖锐

## 模块 5：味美思与苦精

### 5.1 味美思 (Vermouth)
| 类型 | 风味 | 经典用途 |
|------|------|---------|
| Dry (干) | 草本、清冽 | Dry Martini |
| Sweet (甜) | 香料、焦糖 | Manhattan |
| Bianco (白) | 花香、微甜 | 不那么干也不那么甜的Martini变体 |
| Extra Dry | 几乎无味 | 极干 Martini |

> ⚠️ 开瓶后的味美思必须冷藏，2-4 周内用完。过期味美思是马天尼变难喝的头号杀手。

### 5.2 苦精 (Bitters)
| 品牌 | 风格 | 用途 |
|------|------|------|
| Angostura (安高天娜) | 香料、丁香 | 古典、曼哈顿、各种经典 |
| Peychaud's | 茴香、樱桃 | Sazerac |
| Orange Bitters | 柑橘 | Dry Martini 升级 |
| Celery Bitters | 芹菜 | 血腥玛丽、Savoy 系列 |
| Chocolate Bitters | 可可 | Old Fashioned 变体 |
