## 模块 1：高级技法

### 1.1 干摇 (Dry Shake) 与反向干摇 (Reverse Dry Shake)
```
用途：制作丰厚泡沫（酸酒类加蛋清）

标准干摇：
  1. 原料 + 蛋清入壶，不加冰
  2. 用力摇 15 秒（乳化蛋清）
  3. 加冰，再摇 15 秒（降温）
  4. 双重过滤（Hawthorne + 细滤网）滤入杯中

反向干摇（更细腻的泡沫）：
  1. 原料 + 冰入壶，先正常摇 15 秒
  2. 滤出冰块
  3. 无冰再摇 15 秒（打泡沫）
  4. 双重过滤
```

### 1.2 洗杯 (Washing/Rinsing)
```
用途：给杯壁附着一层薄薄的风味

方法：
  1. 少量风味酒（如苦艾酒、Islay威士忌）倒入杯中
  2. 旋转杯身，让酒液挂满杯壁
  3. 倒掉多余液体
  4. 倒入主酒

经典应用：
  - Sazerac：苦艾酒洗杯
  - Martinez：Islay威士忌洗杯
  - Dirty Martini：橄榄汁洗杯
```

### 1.3 烟熏技法
```
方法一：烟熏枪
  1. 调好的酒倒入杯中
  2. 玻璃钟罩罩住
  3. 烟熏枪注入烟雾（木屑：苹果木/樱桃木/泥煤）
  4. 等待 30-60 秒让风味附着

方法二：火焰熏杯
  1. 在岩石杯中点燃肉桂条或迷迭香
  2. 迅速用杯垫盖住，让烟雾充满杯腔
  3. 停 10 秒，揭盖，倒入酒液
```

### 1.4 分子调酒入门
| 技术 | 原理 | 应用 |
|------|------|------|
| 球化 (Spherification) | 海藻酸钠 + 氯化钙反应 | 果汁"鱼子酱" |
| 泡沫 (Espuma) | 虹吸瓶 + 大豆卵磷脂 | 鸡尾酒泡沫 topping |
| 凝胶 (Gel) | 琼脂/明胶 | 酒精果冻 shot |
| 真空低温 (Sous-vide) | 恒温水浴 | 风味浸泡 |

## 模块 2：风味搭配理论

### 2.1 风味轮搭配法
```
搭配策略：
  1. 相邻搭配（安全）：柑橘 + 柑橘 → Orange + Lime
  2. 对角搭配（经典）：甜 + 苦 → 糖 + 金巴利
  3. 桥接搭配（进阶）：香草桥接水果与香料 → 香草 + 草莓 + 黑胡椒
  4. 对撞搭配（大胆）：咸 + 甜 → 培根脂肪洗波本 + 枫糖浆
```

### 2.2 风味搭配速查表
| 基酒 | 最佳水果搭档 | 最佳香料搭档 | 最佳草本搭档 |
|------|-------------|-------------|-------------|
| 金酒 | 柑橘、莓果 | 杜松子、豆蔻 | 迷迭香、百里香 |
| 伏特加 | 任意（画布） | 任意 | 任意 |
| 朗姆 | 菠萝、青柠 | 肉桂、丁香 | 薄荷、罗勒 |
| 龙舌兰 | 葡萄柚、青柠 | 辣椒、孜然 | 香菜、龙舌兰草 |
| 波本 | 桃子、樱桃 | 香草、肉桂 | 鼠尾草 |
| 干邑 | 梨、杏 | 肉桂、八角 | 薰衣草 |

### 2.3 脂肪洗 (Fat Washing)
```
原理：酒精溶解脂肪中的风味 → 冷冻让脂肪凝固 → 去除脂肪 → 留下风味

步骤：
  1. 选脂肪（培根油、黄油、椰子油、芝麻油）
  2. 加热至液态
  3. 倒入基酒中搅拌
  4. 室温静置 2-4 小时
  5. 冷冻过夜（脂肪上浮凝固）
  6. 捞出脂肪层，过滤酒液

经典应用：
  - 培根脂肪洗波本 → Bacon Old Fashioned
  - 棕黄油洗波本 → Brown Butter Bourbon Sour
  - 芝麻油洗金酒 → Asian Gin & Tonic
```

## 模块 3：酒单设计

### 3.1 酒单结构
```
标准酒单构成：
  ┌─ Signature (签名款) × 3-5
  │   └ 酒吧的灵魂，辨识度最高
  ├─ Classics (经典) × 5-8
  │   └ 安全牌，客人信赖
  ├─ Seasonal (季节) × 3-4
  │   └ 限时制造紧迫感
  ├─ Low-ABV / No-ABV × 2-3
  │   └ 照顾不喝酒的客人
  └─ Highballs / Beer & Wine × 3-5
      └ 快速出品，提高翻台率
```

### 3.2 酒单定价公式
```
售价 = (原料成本 × 4) ~ (原料成本 × 5)
举例：
  Negroni 原料成本 ≈ ¥12
  售价 ≈ ¥48-60

原料成本计算：
  金酒 30ml @ ¥0.8/ml = ¥24
  金巴利 30ml @ ¥0.5/ml = ¥15
  甜味美思 30ml @ ¥0.3/ml = ¥9
  总原料 = ¥48... 不对，这是酒吧进价算

  更准确：用整瓶售价除以杯数
  金酒 ¥180/瓶 ÷ 25杯 = ¥7.2/杯
  金巴利 ¥120/瓶 ÷ 25杯 = ¥4.8/杯
  甜味美思 ¥90/瓶 ÷ 25杯 = ¥3.6/杯
  Negroni 原料成本 = ¥15.6
  售价 = ¥62-78
```

### 3.3 备料表 (Prep List)
```
每日开吧前：
  □ 鲜榨柑橘汁（柠檬/青柠/橙）— 不可隔夜
  □ 糖浆（1:1 热溶法，可批量做3天量）
  □ 酸修饰液（Acid Adjusted Juice）— 可做2天量
  □ 装饰预切（橙皮轮、柠檬扭、樱桃穿串）
  □ 冰块检查（大冰/标准冰/碎冰）
  □ 味美思开瓶检查（闻一下有无氧化）
```

## 模块 4：赛事准备

### 4.1 IBA 赛事分类
| 赛事 | 特点 |
|------|------|
| WCC (World Class Competition) | Diageo 主办，全球最具影响力 |
| IBA World Cocktail Championship | 国际调酒师协会官方赛事 |
| Tales of the Cocktail | 新奥尔良，行业盛会 |
| 国内赛 | 调酒师大赛、金酒大赛等品牌赛 |

### 4.2 参赛配方设计流程
```
1. 解读赛题（基酒限定？主题？）
2. 灵感来源（故事线/地方文化/个人经历）
3. 原型开发（3-5个方向，快速试做）
4. 精修配方（10-20轮迭代）
5. 命名与故事包装
6. 出品设计（杯具、装饰、温度、出酒方式）
7. 演示练习（时间控制、话术、动作编排）
8. 压力测试（嘈杂环境/紧张状态下稳定出品）
```
