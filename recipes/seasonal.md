# 季节特调配方 (Seasonal)

> 根据春 / 夏 / 秋 / 冬四季食材与情绪设计的专属配方，以及节假日特调。

---

## 🌸 春季配方 (Spring)

### 1. Sakura Sour
```
情绪关键词：重生、柔软、期待

配方：
  伏特加              45ml
  樱花利口酒(Sakura)   15ml
  青柠汁               20ml
  蜂蜜                 10ml
  柚子花水(Orange Blossom Water)  3 drops
  蛋清                 1个

技法：干摇+加冰摇
装饰：盐渍樱花 + 柚子皮
杯具：Coupe
应季：3-5月（樱花季）
```

### 2. Peony Collins
```
配方：
  金酒                40ml
  接骨木花利口酒       20ml
  白桃泥               30ml
  柠檬汁               20ml
  玫瑰气泡水           80ml

技法：摇和前四种→补气泡水
装饰：玫瑰花瓣 + 白桃片
```

### 3. Spring Cleaning
```
配方：
  金酒                45ml
  黄瓜汁               30ml
  青柠汁               15ml
  薄荷糖浆             10ml
  苏打水               80ml

技法：摇和前四种→补苏打水
概念：新年"大扫除"，清新的开始
```

## ☀️ 夏季配方 (Summer)

### 4. Classic Mojito（完美版）
```
情绪关键词：解放、冰凉、热带

配方：
  白朗姆              50ml
  青柠汁               30ml
  糖浆                 20ml
  薄荷叶               8-10片（拍醒不捣碎）
  苏打水               80ml

技法：
  1. 薄荷叶放入手心拍醒（释放精油）
  2. 加糖浆和青柠汁
  3. 不捣碎薄荷（避免苦味）
  4. 加满碎冰
  5. 朗姆酒 → 苏打水
  6. 用吧勺上下提拉混合

常见错误：把薄荷捣烂 → 整杯发苦
```

### 5. Hugo Spritz
```
配方：
  普罗塞克起泡酒       100ml
  接骨木花糖浆          30ml
  苏打水               80ml
  薄荷叶               5片
  青柠片               2片

起源：意大利阿尔卑斯地区
技法：兑和，大杯满冰
酒精含量：低（适合海量饮用）
```

### 6. Watermelon Margarita
```
配方：
  龙舌兰 Blanco        50ml
  西瓜汁               60ml
  青柠汁               20ml
  糖浆                 10ml
  海盐                 少许

技法：摇和（西瓜出水多，摇和充分乳化）
装饰：西瓜角 + 青柠边盐圈
关键：西瓜选无籽的，鲜榨汁
```

### 7. Coconut Gin Fizz
```
配方：
  椰子油洗金酒          50ml
  柠檬汁                25ml
  椰子奶油(Coconut Cream)  15ml
  蛋清                  1个
  苏打水                80ml

技法：干摇+加冰摇→补苏打水
关键：椰子油洗（见 techniques/advanced.md）
```

### 8. Aperol Spritz（意式开胃）
```
配方：
  阿佩罗(Aperol)        60ml
  普罗塞克              90ml
  苏打水                60ml
  橙片                  1片

技法：兑和，大杯满冰
顺序：苏打水→普罗塞克→阿佩罗（让颜色分层更美）
```

### 9. Painkiller（海岛经典）
```
配方：
  潘趣酒(Pusser's Rum)  50ml
  菠萝汁                80ml
  椰子奶油              30ml
  橙汁                  30ml

技法：摇和（不加冰入杯，热带风格常不过滤）
装饰：肉豆蔻粉 + 樱桃
起源：英属维尔京群岛，Virgin Pina Colada 的加强版
```

## 🍂 秋季配方 (Autumn)

### 10. Pumpkin Old Fashioned
```
情绪关键词：温暖、怀旧、丰盛

配方：
  波本威士忌            50ml
  南瓜派糖浆             15ml
  安高天娜苦精           2 dashes
  墨西哥青柠皮           1条
  丁香                   2粒

技法：搅拌
杯具：古典杯
关键：南瓜派糖浆自制（南瓜泥+肉桂+丁香+红糖+奶油）

自制南瓜派糖浆配方：
  南瓜泥 100g + 红糖 150g + 水 200ml
  + 肉桂棒 1根 + 丁香 3粒 + 肉豆蔻 1撮
  煮 20 分钟 → 过滤 → 冷存可用 2 周
```

```
配方：
```

```
  柠檬汁                 20ml
```

### 15. Mulled Wine（热红酒）
```
配方：
  红酒（不要太贵）        750ml
  橙子                     1个（切块）
  肉桂棒                   2根
  八角                     3个
  丁香                     6粒
  糖/蜂蜜                  50-100g
  白兰地（可选）           30ml

做法：
  1. 所有材料入锅
  2. 小火加热至 70-80℃（不要沸腾！）
  3. 维持 20 分钟（不要超过 40 分钟，会变苦）
  4. 过滤装杯

关键：红酒选年轻的不要太涩的
替代：苹果酒版本（Apple Cider Mulled）
```

## 🎄 节日特调

### 18. Christmas Morning（圣诞晨酒）
```
配方：
  伏特加                  40ml
  热浓缩咖啡              60ml
  白可可利口酒            20ml
  薄奶油(foam)            15ml
  肉桂粉                  少许

技法：摇和→顶层薄奶泡沫
装饰：肉桂棒 + 可可粉
场景：12月25日早晨
```

### 19. New Year's Eve Champagne Cocktail
```
配方：
  方糖                   1块
  安高天娜苦精            2 dashes
  白兰地                 15ml
  香槟                   补满

技法：
  1. 方糖放古典杯底
  2. 苦精滴在方糖上
  3. 加白兰地
  4. 轻轻倒入冰镇香槟
  5. 橙皮装饰
```

### 20. Thanksgiving Punch
```
配方（大批量）：
  波本威士忌              500ml
  苹果汁(未过滤)          500ml
  蔓越莓汁                250ml
  青柠汁                  100ml
  枫糖浆                  100ml
  苏打水                  500ml
  新鲜蔓越莓               100g
  苹果片                   若干

大批量做法：
  1. 前五种混合，大容器冷藏
  2. 上桌前加苏打水（保持气泡）
  3. 蔓越莓和苹果片装饰
  4. 每杯约 150ml

比例：1 份酒基底 + 1 份苏打水
```

## 季节配对速查表

| 季节 | 推荐温度 | 推荐基酒 | 推荐风味 | 禁忌 |
|------|---------|---------|---------|------|
| 春 | 5-8°C | 金酒、伏特加 | 花香、果香 | 过于浓郁的 |
| 夏 | 2-5°C | 朗姆、龙舌兰 | 热带、柑橘 | 热香料、重奶油 |
| 秋 | 8-12°C | 威士忌、白兰地 | 香料、南瓜 | 过于冰镇的 |
| 冬 | 10-15°C | 威士忌、波本 | 温暖、烟熏 | 清凉的碳酸饮 |

> ⚠️ 适量饮酒，禁止酒驾。热饮酒注意温度不要烫伤。
