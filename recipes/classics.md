# 经典鸡尾酒配方库 (Classics)

> 收录 30 款经过时间检验的经典配方，附技法、变体与品鉴要点。
> 来源参考：IBA 官方标准 + Death & Co + Savoy Cocktail Book

---

## 一、Sour（酸酒）家族

### 1. Whiskey Sour
```
风格：酸酒 | 起源：约1860s | 技法：摇和

配方：
  波本威士忌        60ml
  柠檬汁            30ml
  简单糖浆           15ml
  蛋清（可选）       1个

技法：加蛋清则干摇15秒→加冰摇15秒→双重过滤
杯具：Coupe 或岩石杯
装饰：橙皮轮 + 鸡尾酒樱桃
变体：Boston Sour（加整块威士忌杯垫）
```

### 2. Daiquiri
```
配方：
  白朗姆          60ml
  青柠汁          25ml
  简单糖浆         15ml

技法：摇和，加冰滤入冰镇 coupe 杯
装饰：无（或仅一青柠皮扭）
品鉴：入口清冽，酸甜交织，回味干爽
变体：Havana Club Especial 替代白朗姆
```

### 3. Margarita
```
配方：
  龙舌兰 Blanco    50ml
  君度            20ml
  青柠汁          20ml
  盐边（可选）     1圈

技法：摇和，冰镇 coupe 杯
装饰：青柠角
关键：必须是鲜榨青柠汁，柠檬汁不行
变体：看 recipes/modern.md（数百种）
```

### 4. Aviation
```
配方：
  金酒            50ml
  樱桃白兰地(Marashino)  15ml
  柠檬汁           20ml
  紫罗兰利口酒(Crème de Violette) 5ml

技法：摇和，滤入 coupe
装饰：柠檬皮
注：紫罗兰利口酒是争议点，可省
```

### 5. Pisco Sour
```
配方：
  皮斯科白兰地      60ml
  青柠汁           30ml
  糖浆             20ml
  蛋清             1个
  安高天娜苦精      2 dashes（顶层装饰）

技法：干摇+加冰摇，双重过滤
杯具：Coupe
起源：秘鲁/智利争议
```

## 二、Manhattan（马天尼型）家族

### 6. Manhattan
```
配方：
  黑麦威士忌        50ml
  甜味美思          25ml
  安高天娜苦精      2 dashes

技法：搅拌，滤入冰镇 coupe 或 Nick & Nora
装饰：鸡尾酒樱桃
变体：Perfect（甜+干各半）/ Dry / Perfectly Balanced
```

### 7. Dry Martini
```
配方：
  金酒            60ml
  干味美思        10ml
  橄榄或柠檬皮     1枚/1条

技法：搅拌，25-30秒
杯具：马天尼杯（冰镇）
注：橄榄盐水(volume盐水)是合理变体
争议：搅拌vs摇和？标准答案是搅拌
```

### 8. Negroni
```
配方：
  金酒            30ml
  金巴利          30ml
  甜味美思        30ml

技法：搅拌，滤入冰镇 rocks 杯
装饰：橙皮（火焰更佳）
变体：白尼格罗尼(全部换成白苦艾酒替代)
```

### 9. Boulevardier
```
配方：
  波本威士忌        30ml
  金巴利           30ml
  甜味美思          30ml

技法：搅拌，rocks 杯加方冰
装饰：橙皮
别名：咖啡师的尼格罗尼
```

### 10. Vesper
```
配方（邦德原创，2007年才公开）：
  金酒            45ml
  伏特加           15ml
  Lillet Blanc     15ml（原作是 Kina Lillet，已停产）

技法：搅拌
杯具：冰镇 coupe
注：Kina Lillet 停产后，Vesper 配方有争议
```

## 三、Highball（海波）家族

### 11. Gin & Tonic
```
配方：
  金酒            45ml
  汤力水          120ml
  青柠角           1个

技法：兑和（沿杯壁注入保持气泡）
杯具：高球杯
关键：汤力水必须是冰的，青柠挤汁后投入
变体：添加苦精 / 黄瓜条 / 迷迭香
```

### 12. Rum & Coke
```
配方：
  白朗姆          50ml
  可乐           120ml
  青柠角           1个

技法：兑和，高球杯
装饰：青柠
```

### 13. Moscow Mule
```
配方：
  伏特加          50ml
  青柠汁          15ml
  姜汁啤酒        120ml

杯具：铜马克杯（传统）
技法：兑和
关键：铜杯保持饮品冰凉，增强气泡感
```

### 14. Dark 'n' Stormy
```
配方：
  黑朗姆（高年份）  50ml
  姜汁啤酒         120ml

技法：兑和，高球杯
关键：黑朗姆在上，姜汁啤酒在下（风暴感）
禁忌：不允许用哥塞特(Goslings)，那是调好了的预调
```

### 15. Tinto de Verano
```
配方：
  红苦艾酒         100ml
  柠檬汽水(Lemonade) 100ml

起源：西班牙夏日饮品
变体：加一片柠檬
```

## 四、Old Fashioned（古典）家族

### 16. Old Fashioned
```
配方：
  波本威士忌       60ml
  简单糖浆          10ml（可选，1块方糖等效）
  安高天娜苦精      2 dashes
  橙皮              1条

技法：调酒杯中溶解糖→加苦精→加冰→搅拌→滤入方冰古典杯
装饰：橙皮（火焰）
关键：30秒以上搅拌，充分稀释
```

### 17. Sazerac
```
配方：
  黑麦威士忌（或裸麦）  60ml
  简单糖浆               10ml
  安高天娜苦精           2 dashes
  佩丘德苦精             2 dashes
  苦艾酒（洗杯）          少许

步骤：
  1. 苦艾酒洗古典杯，倒掉
  2. 其余材料加冰搅拌
  3. 滤入已洗杯的古典杯
  4. 柠檬皮装饰
```

### 18. Mint Julep
```
配方：
  波本威士忌       60ml
  薄荷叶           8-10片
  简单糖浆          10ml
  碎冰             满杯

技法：捣碎轻拍薄荷→加糖浆和威士忌→满碎冰→用吧勺提拉
杯具：银质 Julep 杯（传统）
装饰：大量薄荷叶
赛事：肯塔基赛马会的官方指定饮品
```

## 五、Long Island（长岛）家族

### 19. Long Island Iced Tea
```
配方（经典版）：
  伏特加          15ml
  金酒            15ml
  白朗姆          15ml
  龙舌兰          15ml
  杜松子酒(Countreau)  15ml
  柠檬汁           25ml
  糖浆             15ml
  可乐（补满）      少许

技法：兑和，高球杯满冰
装饰：柠檬片
注：咖啡因+酒精，适量饮用
```

### 20. Tokyo Iced Tea
```
配方：Long Island + Midori（绿）
```

### 21. California Iced Tea
```
配方：Long Island - 可乐 + 杏子白兰地 + 橙汁
```

## 六、其他经典

### 22. Sidecar
```
配方：
  干邑白兰地       50ml
  君度            20ml
  青柠汁           20ml

杯具：Coupe，糖边可选
技法：摇和
```

### 23. French 75
```
配方：
  金酒            30ml
  柠檬汁           15ml
  糖浆             10ml
  香槟            补满

技法：摇和前三种→倒入香槟杯→补满气泡水
杯具：香槟杯或高球
命名：法国75毫米野战炮
```

### 24. Between the Sheets
```
配方：
  白兰地          30ml
  白朗姆          30ml
  君度            20ml
  柠檬汁           15ml

技法：摇和
杯具：Coupe
别名：Sidecar 的双基酒版本
```

### 25. Bee's Knees
```
配方：
  金酒            60ml
  柠檬汁           20ml
  蜂蜜糖浆（2:1）  20ml

技法：摇和
杯具：Coupe
命名：大萧条时期俚语（最好的）
注：蜂蜜需先用热水稀释成糖浆
```

### 26. Last Word
```
配方：
  金酒            30ml
  绿查特酒(Chartreuse)  30ml
  马拉希诺樱桃酒       30ml
  青柠汁             30ml

技法：摇和
杯具：Coupe
命名：1920s 底特律酒店同名鸡尾酒
特点：四种原料1:1:1:1，极其平衡
```

### 27. Paper Plane
```
配方：
  波本威士忌        30ml
  Aperol            30ml
  奎门娜苦艾酒       30ml
  青柠汁             30ml

技法：摇和
杯具：Coupe
命名：灵感来自歌手 MV 中的包裹纸飞机
```

### 28. El Diablo
```
配方：
  龙舌兰 Blanco      45ml
  黑加仑利口酒        15ml
  青柠汁             15ml
  姜汁啤酒           60ml

技法：摇和前三种→补满姜汁啤酒
杯具：高球杯
```

### 29. Jungle Bird
```
配方：
  黑朗姆           45ml
  坎帕利(Campari)   15ml
  黑加仑利口酒      10ml
  青柠汁            30ml
  糖浆              15ml

起源：1970s 吉隆坡
特点：苦+甜+酸的完美热带平衡
```

### 30. Penicillin
```
配方：
  调和苏格兰威士忌   60ml
  柠檬汁             20ml
  生姜蜂蜜糖浆        20ml
  艾莱岛威士忌(烟熏)  10ml（顶层）

步骤：
  1. 前三种摇和，滤入方冰古典杯
  2. 顶层轻轻淋上艾莱岛威士忌（不混合）
  3. 柠檬皮装饰，置于顶层

创作者：Sam Ross，纽约 Milk & Honey
```

## 经典配方使用指南

| 场合 | 推荐款 | 理由 |
|------|--------|------|
| 餐前开胃 | Dry Martini, Negroni, French 75 | 干爽，激发食欲 |
| 餐后消化 | Limoncello Sour, Sidecar, Amaro Sour | 帮助消化 |
| 夏日户外 | Gin & Tonic, Mojito, Dark 'n' Stormy | 清爽解渴 |
| 冬日晚间 | Old Fashioned, Hot Toddy, Sazerac | 温暖、浓郁 |
| 甜渣友好 | Bee's Knees, Aviation, Last Word | 平衡圆润 |
| 男生/烈性 | Old Fashioned, Manhattan, Sazerac | 醇厚有力 |

> ⚠️ 适量饮酒，禁止酒驾，未成年人请以无酒精版本练习。
