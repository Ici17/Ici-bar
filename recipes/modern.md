### 17. Shrubs（醋饮）
```
基础糖醋配方（可做苏打水基底）：
  水果/香料 250g + 白糖 200g + 苹果醋 250ml
  混合，静置室温 3 天，冷藏可存 2 周

配方示例 — 草莓醋饮：
  草莓醋饮           45ml
  苏打水             120ml
  柠檬汁             15ml
  新鲜草莓           3颗

技法：兑和
```

### 18. Clarified Milk Punch
```
经典配方（18世纪技术，2010年代复兴）：
配方：
  波本威士忌         60ml
  柠檬汁             30ml
  糖浆               20ml
  全脂牛奶           60ml

技法：
  1. 全部混合摇和
  2. 静置 30 分钟（蛋白质絮凝）
  3. 细滤纸过滤（极慢，约需 1 小时）
  4. 得到乳白色透明液体

结果：丝绒口感但视觉透明的神奇效果
创作者：Jamie Moyer，酒吧名 Mibar（纽约）
```

## 六、脂肪洗风味酒 (Fat-Washed)

### 19. Bacon Fat Bourbon Old Fashioned
```
配方：
  培根脂肪洗波本      60ml
  枫糖浆               10ml
  安高天娜苦精         2 dashes
  橙皮                 1条

培根脂肪洗步骤：
  1. 培根油 30g 加热融化
  2. 倒入波本威士忌 200ml，搅拌
  3. 室温静置 3 小时
  4. 冷冻过夜（脂肪上浮凝固）
  5. 用咖啡滤纸过滤
```

## 现代配方创作指南

```
好配方的标准（MixMaster 大师标准）：

1. 可复制性：任何调酒师按你的配方做，味道相同
2. 可获得性：原料在 80% 的城市能找到
3. 可记忆性：命名有意思，客人能记住
4. 可进化性：留有变体的空间
5. 有故事：不是最好的酒，但有最好的故事

创作检查清单：
  □ 我解决了一个真实的味觉需求吗？
  □ 这个组合有没有先例？有多少？
  □ 如果去掉一种原料，这杯酒还成立吗？
  □ 这杯酒适合在什么场合喝？
  □ 它有让人想再喝一杯的理由吗？
```

> ⚠️ 适量饮酒，禁止酒驾。
