# 🍸 Bartender Agent

[![OpenClaw](https://img.shields.io/badge/OpenClaw-Skill-blue)](https://openclaw.ai)
[![SkillHub](https://img.shields.io/badge/SkillHub-Install-green)](https://skills.sh)

> 从零基础到大师级的调酒师 Agent，完整的鸡尾酒教学体系

## ✨ 特性

- 📚 **五级课程体系**：小白 → 初级 → 中级 → 高级 → 大师
- 🍹 **100+ 配方库**：经典鸡尾酒、现代创新、季节限定、大师签名款
- 🎯 **系统化教学**：技法 → 原料 → 设备 → 实战案例
- 🏆 **赛事级内容**：花式调酒、比赛规则、酒单设计
- 📖 **中文原创**：完整中文教程，适合国内学习者

## 🚀 快速开始

### 安装

```bash
npx skills add Ici17/Ici-bar
```

### 使用

安装后，Agent 会自动识别调酒相关话题。你可以问：

- "教我调一杯 Old Fashioned"
- "什么是六大基酒？"
- "帮我设计一个春日酒单"
- "摇酒和搅拌有什么区别？"

## 📚 课程内容

### Level 1 - 小白入门
- 吧台工具认知
- 四大基础技法（Shake/Stir/Build/Muddle）
- 5 款入门配方

### Level 2 - 初级调酒师
- 六大基酒体系
- 15 款经典配方
- 基础装饰技法

### Level 3 - 中级调酒师
- 配方解构与平衡
- 风味轮与搭配
- 分层与烟熏技法

### Level 4 - 高级调酒师
- 分子调酒入门
- 酒单设计原理
- 赛事规则与准备

### Level 5 - 大师级
- 创作哲学
- 酒吧运营管理
- 行业文化与趋势
